红米2不再是唯一的选择,大神F1体验

829 1




千元机我们看的太多了，但是至少在我眼中，基本上所有的厂家在宣传文案中都把自家的千元机写的很美好。弄的各种功能都有，还是性能怪兽并且又低价，但是在我上手之后每次都有有一种落差。感觉很多千元机就是应该值这个价钱。并没有感觉觉得我买划算了。但是回头一想现在的手机不都是一分钱，一分货么？那怎么会有厂家傻逼逼的给高配不赚钱呢？对吧。

但是随着互联网手机的急速发展，低价高配的手机也逐渐越来越多。所以之前优势明显的红米细节已经开始被追赶，就在之前红米2发布之后被大家吐槽的1G RAM，却引来了“少装应用，还是很流畅的”说法。在此笔者只能说对小米说，现在的消费者不傻。


究竟有没有既能装APP又能流畅运行的699元手机呢？答案是有的！大神最近推出了大神F1就是其中一款。这款2G RAM的手机究竟能不能秒小米呢？体验正式开始。


外观 · 699

大神F1采用了上下黑白的熊猫配色，机身的正面让我一眼看上去脑海里瞬出现了“魅族”这两个字，真正的元凶就是机子下部的小圆点。看来经典的设计是会不断的被模仿。但是谈超越？大神F1还远远不及这个程度。因为这个小圆点虽然也有背光灯，但是仅仅只能作为Home键来使用，在功能性是上远不如魅族的来的全面，也不能当做提示灯来使用。唯一值得称赞的是作为一款699的手机F1的按键是带背光的。





对应顶部的听筒位置是和屏幕的边框相连接的，这样有利于清洁在平时的使用过程中并不会积累灰尘。

大神F1的屏幕是采用全贴合的工艺（貌似现在699的手机几乎都采用了全贴合了，不错的趋势）。所以在息屏的时候就不会在出现大灰屏的情况了。在大神F1息屏的时候虽然没有像旗舰手机黑的那么彻底但是看上去也是会给人一体的感觉。




但是亮屏之后千元机常见的标准化边框再一次出现在我的眼前，相比于魅蓝非标准的ID设计，大神F1给我的感觉会显得略微普通一些，和我们日常见到的千元机在设计上并没有什么提升的地方。

大神F1的后盖采用的是全包裹的设计，电源键和音量键被安放到了机身的左右两侧。5寸屏幕的F1电源键设计在右上角是比较舒适的一个位置，大拇指正好能够覆盖的到。按键都是为塑料的材质，按键的手感适中反馈明确。



F1的后盖采用了类似类肤质的涂层，摸上去确实和普通工程塑料相比起来会好上不少。手感也好很多了。这点是F1加分的地方。毕竟手机摸的舒不舒服也是人们感知的一台手机好坏的重要一环。





在开箱的时候我们就有提到过F1在镜头的位置上加了一圈的金属圈来保护这个镜头盖。使其在日常的使用中不会被刮花，确实啊有个这个圈镜头盖就不会直接和桌面直接接触了。使用了一段时间下来也没有出现划痕。



而在镜头的对称的底部则是为扬声器的位置，我始终不理解为什么大神会把扬声器放在这个位置上，因为只有一旦手机正面朝上时扬声器必定会被压的死死的。别说听歌呢有时候来电铃声都会错过，这一点是需要改进的。

并且在后盖上有两个LOGO，一个是大神的还有一个比较小的是酷派的。这样的做法让我始终觉得大神一直无法摆脱酷派的影子，这样是不利于大神这个品牌的建设的。




大神F1的整体外观采用了标准化的设计，在工艺难度上并不复杂所以制造良率也比较的高，所以基本上不会产生一机难求而的情况。并且再加上手感不错的后盖，就使得F1有了那么一些矜持。但是大神的内涵不仅仅在表面上，我们接着往下看。


屏幕 · 色彩

我们提到过千元机的屏幕现在进步十分的明细，尤其在工艺提升到全贴合之后显示效果相比于之前可以说是提高了不止是一点点啊。


而大神F1采用了5寸的屏幕，虽然分辨率只有1280x720但是在日常使用中已经是完全够用了。OK我们现在就来看一下大神F1屏幕的具体表现。

通过以上和魅蓝手机的对比，F1在屏幕的色彩发色倾向上和魅蓝形成了明显的差异两台换手机显示完全不同的风格，F1的发色会更浓艳一些而魅族会稍微淡那么一点，这个没有好坏见仁见智咯。

并且F1的屏幕是偏冷色的，所以在显示灰色或是白色的图片下它并不会出现偏品红的现象。





两台手机在显示黑色的背景下，并无发现漏光的问题。这里需要提到的就是可视角度的问题。因为官方称F1的可视角度可以达到178°。



那么在实际的测试中F1屏幕的可视角度确实十分的优秀，在大角度的情况下也还是能够依然看保持基本的屏幕不偏色的问题。这一点是需要点赞的。


2G RAM · 性能

2G RAM是这次大神F1的集中卖点，因为在699的这个价格区间内几乎都是采用了1G RAM 的手机，我们都知道手机内存需要越大是越流畅的，所以在之前的很多699元的手机中他们的外观都十分的赤出色，但是唯独就是在整体的流畅度上不如人意，其实更多的是因为成本的限制所致。这一次大神可以说是花了钱的。

F1采用了晓龙410的SoC，这颗处理器是高通64位的处理器但是F1现在的系统版本还是安卓4.4.4所以在日后的升级5.0之后系统整体运行速度还会在提升一个档次。


但是在现在这个版本的使用中，F1的整体流畅性是相当不错的。为什么呢？我们发现F1的COOL UI 相比于大神X7的UI会相对做了不少的减法，就使得整个系统更轻了。再加上2G RAM的支持甚至我可以毫不犹豫的说 F1在整体运行的流畅度上已经超过了大神X7双网通的版本了。并且这个2G RAM是采用了LPDDR3 的存储，效率和运行上会更加有优势。

所以F1的的确确的利用软硬结合的方式，让699元的手机不在卡顿。流畅性这点在我们日常使用中也是十分关键重要的一个环节。大神把钱集中花在这个方面。


Cool UI · 减法

我们刚刚提到过F1为了保证系统整体的流畅度把Cool UI做了一些减法，但是不是什么都没有了呢？

并非是这样，F1的系统方面仅仅简化了视觉方面的一些东西就一方面降低了系统本身的内存占用量还有一方面也减轻了处理器对于画面渲染的强度。


就例如拨号界面我们看到的是单色的画面，短信界面也是一样。设置界面从原来的两页被集成到了一页上，还有下拉菜单的简化。这些都是很明显的优化减法的过程，效果也是十分的简单粗暴有效。






在功能性方面Cool UI 并没有缩减任何一项功能，如果是Cool UI长期用户这点你们是可以放心的。


成像
大神F1在相机方方面采用了F2.2的光圈，并且在感光元件上采用了1.4μ并且使用了800万的像素，好了参数交代完毕接下来我们直接看样张。


白天成像

这次的对比机型我选择了F1的直接竞争对手魅蓝手机

通过第一张样张对比，我们可以看待F1在色彩的饱和度上相对于魅蓝更加的艳丽一些，这样的做法是比较容易讨好人的眼球，而魅蓝是偏向于真实的发色。


从第二张的小花上我们一样可以得出以上的结论，但是我们发现了一个问题就是在这张图片上魅蓝明显是有过曝了，而F1控制的却十分的稳定。在画面的整体结构上轮廓清晰。


第三张样张我们观察地面，在白平衡上魅蓝是有那么一丁点的偏绿的倾向。而F1还原的算是比较的准确，这一点是我比较的惊讶的。


从第四张样张上我们可以一样得出相同的结论。


夜景成像

那么在夜景发面在第一张光线比较充足的情况下，两台手机的成像是差不多的。但是在细节上魅蓝在炫光的控制上会比F1更好一些。


在第二张光线稍暗一些的情况下，魅蓝整体画面的亮度是高于F1的，但是由于魅蓝的快门速度相对慢一些，导致照片糊掉了。


最后一张样张我们可以直观的看出魅蓝的整体的画面亮度是高于F1的，但是可见的噪点非常多，通过样张信息魅蓝的曝光时间为1/10秒而F1的曝光时间为1/16秒。ISO魅蓝会比F1更高一些。所以魅蓝在画面的亮度上会比F1要亮。




总结
大神F1把钱花在了内存和显示屏上，虽然在外观上是有所妥协但是类肤质的后盖还是挽回可一些手感，这也是千元机不可避免的问题。由于成本的限制就会导致有着太多不完美，但是这样的情况有仅仅何至于千元机呢？

但是至少有一点是可以放心，在千元机的范畴中大神F1真的很流畅。这也让大神F1有了和红米，魅蓝直接对坑的资本。
