“笔”较优秀，三星Galaxy Note 5评测

7285 5

正面2.5D玻璃、背面3D曲面玻璃、金属中框、5.7英寸2K屏幕、八核心处理器和4GB RAM，以上说的并不是自封安卓机皇还发布了好几次的小米Note顶配版，而是三星Galaxy Note 5，唯一一款真正可以被称之为Note的手机，同时也是目前真正的Android机皇。

  • 外观设计、机身做工与握持手感


Note 5是三星有史以来推出过的最漂亮的Note系列手机，至于是不是三星有史以来最漂亮的手机则争议颇多，很多人觉得这个头衔应该给同期发布的Galaxy S6 Edge+，但这并不重要，谁也不能否认Note 5也是一款非常漂亮的手机。




在Note 5和S6 Edge+上，三星除了统配的黑、白和金色之外，新增了一个银色，也就是本次我们评测机的配色，三星今年在机身的调色上可谓下了许多苦功夫，在上半年S6和S6 Edge上，三星就做出了可能是最不俗气的金色机身以及最不低调的黑色机身，尤其是黑色的S6/S6 Edge，在变换角度观看的时候会泛出非常漂亮的深蓝色，并且蓝色的饱和度还会随着光线不同的照射角度来变化。


Note 5的黑色款在调色上和S6 Edge的黑色款是一样的，倒是本次新增的银色款更能够吸引目光，Note 5的银色款的色彩变化程度相比于黑色款自然是小了不少，但是同样的，在光线变换角度照射的时候，Note 5的银色机身会折射出一丁点淡蓝色或是淡绿色，让银色款的机身增添了一些冰凉的视觉感受。


但是自然而然的，一个双面玻璃设计的机身是逃不过用户指纹的，不说用多久了，只要手拿上Note 5的机身，就会留下清晰的指印，若是用个半小时一小时，机身更是会变得沾满指纹，甚至有点油乎乎的感觉，对于这个嘛，没办法。

本次我们手上的机器是港行零售版的Note 5，型号为SM-N9200，和国内将要上市的型号完全一样，并且支持双卡双待全网通，能够无障碍使用移动联通电信的2、3、4G网络，至于在淘宝上颇多的台湾版Note 5，在外观上和网络支持上会有一点区别，首先是在背部会有DUOS的双卡双待标志，另外网络支持上去掉了对CDMA，也就是电信网络的支持。最后来一句题外话：香港地区首批发售的Note 5仅有金色和银色，而台湾地区首批发售的Note 5则只提供金色和白色，至于国行嘛……要等到9月才会发售。














另外关于机身的用料，据说三星本次给Note 5和S6 Edge+用上了7000系列的铝合金，硬度会更高，不过手感层面上，Note 5的边框和S6 Edge的边框并没有什么区别，都是非常细腻的哑光表面，不同之处在于Note 5的边框和背部玻璃面板衔接的地方，三星做了两道的斜面，加上即为精致的磨砂和抛光，让金属和玻璃之间的衔接完美顺畅，即便特地用手去摸也不会有任何的刮手感，所以说虽然三星Note 5和小米Note使用了类似的设计，但是三星在Note 5上对于细节的注重，是小米绝对不会具备的。




至于做工啥的，Note 5的接缝、卡槽和按钮的做工基本无可挑剔，但是在正面的传感器和摄像头开口处，首批的Note 5都会有镀膜歪斜的情况出现，挑战着强迫症患者，不过香港的卖场导购都相当负责，顾客对这台机器不满意，直接二话不说再拿一台，啥时候国内也能有这样的服务态度呢。

不过Note 5上有一个用料的小细节让人甚为不解——Note 5的卡槽是塑料制的，而不是金属制，虽然卡槽的表面和机身边框的平整度非常完美，但是还是希望三星能用更坚固的金属来打造卡槽——毕竟作为一台面向预算不是第一考虑标准的消费者的旗舰手机，这点用料的妥协完全不必要吧。

最后就是Note 5精致机身的代价了——不能换电池和不能插入SD卡，机身内存是32GB起跳，所以倒也还好，不会像买16GB iPhone的消费者那般尴尬，而三星对于此的解释是Note 5使用了三星最新的UFS 2.0闪存，该闪存的读写速度甚至可以达到SSD级别，插一张SD卡会影响用户体验。恩……虽不是没有道理，但是我还是想插卡呀。






至于Note系列手机的灵魂，也就是那只S-Pen，这次同样也有改变，笔身顶部的按钮跟日常使用的圆珠笔一样，按下会有令人舒畅的“咔哒”声音并且会弹出，虽然这按钮的手感跟豪华轿车上的按钮手感还是没得比，但是至少也是一个能玩上一个月的小玩意。


至于笔身嘛，还是塑料，表面很光滑，在夏天使用的时候也是用了一会就有油油的感觉，笔身上的功能按钮手感会比Note 3/4的笔身按钮好不少，三星Note 5的手写笔用的是日本Wacom的技术，带有压感检测，简单来说就是更加像真的“笔”了。

  • 屏幕：

作为AMOLED技术的大力推进者，三星从来都会在自家的旗舰手机上用自家最好的AMOLED屏幕，并称之为“Super AMOLED”屏幕，然后系统之中也提供4种不同的显示模式，笔者个人比较偏好色彩准确度最高的“基本”模式，这一模式下Note 5的屏幕色温会在6500K~7000K之间，可能会被许多人觉得是“大黄屏”，喜欢冷一些色温的就用自适应或者AMOLED影院模式，不过色彩准确度自然就打折扣咯。


















和三星自家的Galaxy S6 Edge对比，Note 5的屏幕有更高的亮度，至于基本模式下的色彩还原能力，则和S6 Edge差不多，都比较准确，别的没什么可说的，总之是目前市面上最顶级的手机屏幕之一，可以与LG G4上的量子点IPS屏幕相媲美，在亮度方面Note 5的亮度变化范围非常广，手动模式下可以在1~400nit之间变化，所以即便是夜间窝在被窝里使用，也不会有丝毫的刺眼感觉。

  • 性能和系统：

这次非常相反地，我们对于Note 5的性能表现没有任何的期待——因为它使用了和Galaxy S6一模一样的Exynos 7420处理器，作为目前市面上最强的手机处理器（没错是最强的），Exynos 7420的性能远远超过了被广泛采用的高通骁龙810以及即将被广泛采用的联发科Helio X10（MT6795），三星早前推出Exynos 7420这颗处理器之时，用极其强悍的性能表现吸引了许多人的目光，而本以为Note 5会使用更强悍的Exynos 7422或者Exynos 7430，结果等到上市我们才发现Note 5用的还是7420，只不过换上4GB RAM了而已。


失望归失望，Note 5的各项性能跑分依旧是碾压目前市面上所有其他手机的，所以说绝对性能方面，Note 5性能依旧是市面上最强的（当然处理器和RAM相同的S6 Edge+理论上和Note 5是同一个水平），并且由于机身相比S6更大，所以更利于处理器的散热，所以Note 5上的7420的稳定性会比S6上的好，能够在更长的时间之内维持在高频率下运行。

至于三星的UFS 2.0闪存，同样是目前一个没有对手的东西，目前其他品牌手机上使用的最好的都只是eMMC 5.0而已，在读写速度上都不如UFS 2.0，反应到实际使用之中，就是开应用更快，拍照保存更快。

至于系统体验方面，Note 5自然是没有任何问题咯，虽然之前S6和S6 Edge上面，细心的用户会发现随着三星更新这二者的系统版本，二者的使用流畅度会大幅度下降，反过头来，S6和S6 Edge最流畅的版本反而是很早期的OE4系统，虽然新系统的续航能力会比旧系统强，但是说实话，还是流畅点好呀，至于note 5上三星会不会这么作死，我们也不知道。

在系统方面Note 5只有唯一一个重点，就是S-Pen的应用。


没有笔的Note，根本不能叫Note。


Note 5上面，这支笔变得更加强大了，首先是系统之中，提供了一个黑屏记事本的功能，在待机状态下拔出手写笔，就可以直接开启一个黑色底的记事本界面，笔迹以白色显示，很好地利用AMOLED屏幕的特性来营造“关屏”的效果，或者也可以设定在任意界面下，按住S-Pen的功能键在屏幕任意部位双击弹出快捷记事贴。


在笔尖悬浮的状态下按下笔身上的功能按钮，Note 5屏幕上会弹出一个圆环，最多能够有6个快捷方式，而下方的3个快捷方式是能够用户自定义的，意义并不算很大，但有有总比没有好。


在浏览网页的时候按下S-Pen上的功能键，就会进入文字选定模式，可以直接在屏幕上选定一段文字之后可以复制或者直接分享到微信、微博等等，就像用鼠标在网页上复制文字一样，好用度很高，但是对于动态元素之中的文字是不能直接复制的。




至于三星称之为“随笔输入”的功能，这个功能是在有输入框的时候，将S-Pen悬停在输入框上，就会出现一个小按钮，用笔尖点击按钮之后，会开启一个手写界面，可以使用手写输入，可以识别简体中文和英文，并且在微信之中，随笔输入还可以直接用笔涂鸦并直接以图片的形式发送给对方，虽然并不是特殊的功能，但是能够为单纯的文字聊天加入很多乐趣。


另外，Note 5的手写笔最大的用处还是截图，并且直接在截图上标记书写，或者是截取屏幕的一小部分，虽然听起来是用手指头也能办到的功能，但是实际使用之中，用手写笔能够实现更为精确的截图，用手写笔进行涂画的精度也会远远高于手指头，完成截图之后也不用特地保存就可以直接分享到聊天工具或者社交网络之中，发完了立刻就删除不占内存，好用程度谁用谁知道。至于那个长截图的功能，同样也有不错的使用场景——比如截取一个网页，或者更实用的，截取一大段撕逼聊天记录来发到微博朋友圈什么的。


另外还有同屏多任务，就如上图所示，屏幕上方是浏览的网页，下方是S Note记事本工具，中间的微信气泡是已经最小化的微信，上网记事聊天互相不耽误，在高分辨率的大屏幕上用笔尖精准地操控手机，降低误操作率也提高了操控效率，逼格和生产力分分钟爆表。


最后就是堪称是目前手机上最好用的记事本软件S Note了，S-Pen搭配S Note，可以真正地做到开会不用带纸笔，三星也有提供S Note的拓展功能安装包，可以让S Note非常非常强大，并且S Note还可以和Evernote进行融合同步，同步到电脑上面也很轻松，简直是商务人士的神器，不过感觉许多买Note系列手机的用户，用到手机坏掉也很少将笔拔出来吧。


Note 5的主题功能和S6 Edge没有区别，二者的主题也是直接通用的，目前三星的主题商店了已经提供了Material Design的Nexus风格主题，非常好看，至少比原生的主题好看到不知道哪里去了，不过当然也会有各种粉嫩系的主题给妹子用咯，所以UI这方面，美观度上Note 5是没问题的，至于各种使用便捷度还是很三星的风格，设置项颇多，但是大可不必去理会。




另外就是心率传感器，虽然大多数时候是个鸡肋东西，但是偶尔给自己掐个脉似乎也不错，并且三星也活用这个传感器，在开启前置摄像头自拍的时候，用手指头压上心率传感器，手机就会触发快门，对于大机身来说，是个相当不错的创意，用起来也很自然。


最后笔者想就Note 5的退步方面特别开一段来说，例如这个指纹识别，Note 5的HOME按键面积更大了，指纹录入的速度和识别速度也更快，不过识别率相比于S6 Edge反而有明显的下降，可能就是因为简化录入所导致的吧，所以用起来反而没有S6 Edge的指纹那么好用了。

还有就是Note 5去掉了红外发射模块，所以自然也是不能用Note 5愉快地遥控家里的电器了。


对于Note 5的系统，我们并没有什么惊喜，作为一款旗舰手机，拥有好看的UI和流畅的动画效果是必须的基本素质，另外，三星的S-Pen在Note 5上面达到了一个新的高度，不论是随手记录还是特地用来做会议纪要，或是灵活地操控多窗口或是单纯地用笔代替手指头来在屏幕上进行精确的图片编辑，S-Pen都能够完美地胜任，甚至可以很负责任地说，有了S-Pen之后，Note 5的生产力是所有手机之中最高的，S-Pen目前在功能性和易用性上已经达到了近乎完美的水准，S-Pen的尴尬之处是对于国内大多数购买Note手机的人来说，他们购买Note的原动力只是一块大屏幕而已，S-Pen对他们来说，再好用也是并没有什么卵用的东西。

  • 拍照：

Note 5的拍照硬件和S6 Edge是完全相同的，1600万像素，16:9索尼IMX240传感器，加上光学防抖镜组和相位对焦技术，不过在实拍之后进行对比，我们发现Note 5上的成像调校会比S6 Edge上更加精进，提升了不少锐度，而暗光条件下，Note 5的色彩渲染能力也会比S6 Edge强。


至于新的手动功能方面，Note 5终于加入了快门速度调节、色温调节和RAW格式输出，三星表示这些特性有望在将来通过软件升级推送给S6和S6 Edge，希望到时候S6和S6 Edge的相机成像质量能够和Note 5一样好吧。














就实拍效果来看，Note 5的表现极为优秀，随时随地双击HOME按键启动相机，默认的自动模式下不论白天还是晚上随手拍就有非常准确的白平衡、色彩还原和曝光控制、极高的画面宽容度、细节表现和锐度，噪点控制能力优秀，飞快的对焦和成像速度使得拍照也很是爽快，即使是在夜间，也有非常优秀的成像质量，略显不足的是在夜间拍照的时候镜头对于眩光的控制居然不如以前的旧款三星手机，虽然有时候可以拍出很漂亮的光芒，但是更多时候，眩光还是对画面有着明显影响的。

  • 充电和续航：

Note 5的电池容量甚至比Note 3更小，只有3000毫安时，这个电池容量甚至比一些5英寸的手机还要小，所以对于Note 5的续航表现我们自然不会有太多期待，能用个一天回家还有剩余基本就OK了，天天充电肯定是免不了的，我们的续航测试分为大型游戏、Wi-Fi上网、4G上网、本地高清视频回放、3G通话、音乐播放，每项30分钟，模拟中度到重度的使用情况。


实际测试下来，Note 5的测试剩余电量为69，对于这个容量电池和这个配置的手机来说表现是相当不错，当然了，这个成绩是在禁用了港版系统自带的Google全家桶之后获得的续航成绩，对于新入手国外版但是准备在国内使用的，我们同样建议禁用Google全家桶，因为Google全家桶会频繁地唤醒系统连接网络，而国内的网络环境不允许连接Google，所以自然手机就陷入了一个死循环，造成严重的发热和耗电了。


至于充电速度，Note 5表现则是挺好的，使用原配的快速充电器进行充电，0~100的充电时间为90分钟，不过考虑到Note 5的电池容量那么小，实际上Note 5的充电效率并没有特别高，实际充电9V/1.67A 的输出也和S6一样，论实际的充电效率，会略低于使用联发科的PumpExpress Plus快速充电技术的金立M5以及使用VOOC闪充技术的OPPO R7 Plus，就只能说是正常的快速充电水平，要知道R7 Plus的电池比Note 5大了足足1100毫安时，然后0~100充电的时间还是完全相同的。

至于具体的充电曲线，在前10分钟，使用原厂的充电器就为Note 5充入了接近25%的电量，然后整个充电速度经过了两次放缓，在80%电量之后充电速度就变得相当慢了，估计也是为了保护电池寿命所做的设定吧。

至于无线充电呢？Note 5支持Qi的无线充电，用三星自家的标准型无线充电的话充电速度就慢的感人，甚至慢得我们都不想去测试，虽然用三星的快充无线充电板会提升无线充的速度，三星官方宣称快速无线充电器的0~100充电时间为120分钟，对于无线充电来说是个很了不起的速度，但是那一个无线充的售价可能会在450元左右，相当放肆的售价，这个快速无线充电的逼格代价有点高……

  • 总结：



本次很奇怪的，三星并没有将Note 5作为主推的产品进行销售，而且从港行Note 5的折合4700元人民币的售价来看，甚至是有史以来最便宜和性价比最高的Galaxy Note，纵观全球市场，三星主推的都是更有逼格的S6 Edge+，甚至在欧洲市场，Note 5都根本不上市销售。确实对于很多用户来说，Note 5的S-Pen是不折不扣的鸡肋，三星看来也是明白这一点的。


不过这样对于亚太地区的用户来说反而是一件好事——我们能够用更实惠的价钱，买到一台一定程度上能代替电脑的，更优秀的智能手机，对于真正需要一台好用、强大的智能手机来说，更是一件大好事，因为在5.5~6英寸这个尺寸等级上，三星Galaxy Note 5依旧是最好的智能手机，没有之一。
