黑马旗舰 奇酷旗舰版长测报告

15844 4
伴随着“好奇心改变世界”的口号，奇酷手机诞生了，我其实最开始是被这句口号所吸引，因为我也一直觉得懒和好奇心是世界上的2大创造力和生产力，所以当奇酷手机宣布要开发布会、奇酷手机要面市了的时候，我也默默地关注了下，却未曾想老周一归来就带来了3款产品：奇酷手机青春版、旗舰版和尊享版，而今天我要给大家带来的评测就是定位在中高端的奇酷手机旗舰版。

# 开 箱 #
是否因为老周被人称作“红衣教主”而对手机采用了大红色的包装，这点我们不得而知，书本样式的包装多多少少让我们想起魅族mx系列的传统包装，只是奇酷这本“书”无论是在配色上还是份量上都要来得更加强烈。



既然是看书，我们也得有看书的将就，不要一开始就直奔正文，扉页的内容也许会对你读懂这本书有着至关重要的作用，我们大可仔细品读一番。

感官之美、彩色+黑白夜视双1300万镜头、安全轻快省电的360os，一页一个重点，一个重点就是一个特色，在见到手机真容前我们便对这部手机有了一个初步的认识，而这3点恰恰也是奇酷手机旗舰版最吸引我的地方。







而当奇酷手机旗舰版真的出现在我眼前的时候，我承认它是漂亮的，但不算是有特色的漂亮，因为这个三段式的“大白”设计已经被国产手机玩腻了，我也就见怪不怪，毕竟美的东西大家都喜欢，更何况它真的不难看呢！

拉起预留的拉条，可以很轻易地取出奇酷手机旗舰版，倒也很符合那句“书中自有颜如玉”的词句。





底部的“书本”夹层里，放置的是全部配件，摆放得十分整齐。



这就是奇酷手机旗舰版的全套配件，包括保修卡、说明书、塑料保护壳、高清贴膜、数据线以及充电头。

奇酷旗舰版并没有采用时下热门的Type-C数据线，而是依旧是micro usb接口，但是为了解决黑暗环境下正插反插的问题，奇酷带来了一个小小的黑科技，那就是在数据线的接口一端加了一个小led灯，当我们手持数据线接驳手机的时候，只需要轻轻按下数据线手机端的按钮，幽幽的灯光便可以照亮一个小范围，供我们争取接驳手机。



而关于大家关心的快充功能，很开心地告诉大家，它终于来了！奇酷旗舰版的机身电池容量高达3700mAh，如果依旧用以往的5V2A的输入功率的话不知道充满需要消耗多长时间，庆幸的是我们在奇酷的充电头上看到了9V 2.2A的字样，也就是奇酷旗舰版用上了高通的QC2.0协议，老周把它叫做涡轮闪冲，我承认我还是挺喜欢这个名字的！


360奇酷手机旗舰版正面配备了一块6.0英寸1920*1080分辨率屏幕，有效屏占比达到了83%，外观大小和5.5寸的iPhone6puls基本一致，得益于高屏占比，手机拿在手里并不会让人觉得hold不住。

正面顶部是光线感应器、听筒和800万像素前置摄像头，关于摄像头的体验我们后续拍照篇详细讲解。

正面底部是3颗虚拟触控按键，返回键被放置在了右侧。

机身右侧是电源键和音量调节键，键程舒适，反馈明显。

机身左侧是2个sim卡卡槽，有趣的是奇酷手机旗舰版将它们做了分离设计，并且上面的卡槽是micro sim卡，下面的卡槽是nano sim卡，其实这样的设计还是由他存在的道理的，当你取其中一个卡槽的时候并不会影响到另外一张卡的使用，这也算是一个贴心的设计了。


机身底部是扬声器和通话麦克风，不知道该说奇酷手机实诚还是不懂审美，在当今大家都把通话麦克风做成扬声器对称设计的时候，偏偏回归到了一个单独的小孔，仿佛在告诉用户，我就是麦克风，没必要装扬声器。

顶部仅仅只有一个3.5mm耳机接口，降噪麦克风被放置在了背部摄像头左侧。

奇酷手机旗舰版还有一个细节的设计深得我心，那就是正面屏幕2.5d玻璃的加入，拿在手里的时候会觉得整部手机显得非常圆润，少了一份咯手感，多了一份贴切和讨巧，在光线下更是能让整部手机的质感得到进一步的提升。



# 相 机 #
奇酷手机旗舰版外形上最先吸引大家的，肯定要数其特有的双摄像头了，分别采用了索尼IMX278（RGBW）彩色传感器和索尼Mono黑白夜视传感器，两者均为1300万像素。IMX278 RGBW中的W实际上是每个像素点新增White白色像素点，进光量提升40%，低光环境噪点降低78%，专门负责捕捉彩色影像。Mono黑白夜视传感器负责细节轮廓，比普通彩色传感器多出4倍进光量，当IMX278和Mono配合使用的时候，得到的样张会比普通单个摄像头的样张色彩更加丰富，细节表现更加清晰，夜景表现更为纯净。




好的硬件表现也需要有好的相机app提升体验，奇酷旗舰版的相机app实际操作起来十分方便，主界面采用了左右标签式的设计，默认是普通拍照模式，左滑右滑可以切换不同功能标签，从左到右依次是单反、夜景、摄像、拍照、美颜以及专业6个模式。顶部快捷设置栏囊括了闪光灯开关、HDR开关以及前置后置摄像头切换按钮，点击右上角三个小圆点可以进入其他拍摄模式，其中长曝光模式与专业模式有点重复，其他的5个模式包括全景、水印、GIF、留声拍照以及先拍照后对焦都是我们熟悉的几个模式，使用起来也十分方便，当然可能是由于这几个模式的使用频率相对较低，所以放置在了这个位置。右下角还有9个实时滤镜，提供更多的创作空间。

接下来就让我们来详细认识一下奇酷旗舰版相机的这些不同功能带来的拍摄体验和样张，还原奇酷旗舰版智能双摄的实际拍摄能力。

美颜模式：
在说奇酷手机的美颜模式之前，首先要说一下奇酷相机一个很好玩的功能，那就是模拟年龄的人脸识别功能，在检测到画面中的人脸的情况下，会直接在画面中人脸上方显示预估年龄，当然该功能只是大概预估，实际结果并不准确，老周自己在玩的时候也时常调侃自家的手机把自己照老了。另外值得一提的是，奇酷手机在人脸识别上十分准确，实际测试只要画面中有人脸，奇酷相机都能够准确识别出来。

另外奇酷手机在自拍上，还有一个十分人性化的功能，那就是通过手势来激活拍照，只需要张开手掌，轻轻一握，相机检测到手掌从张开到握住的过程的时候，就会激活拍照，倒数3秒后便会自动释放快门，3秒的时间足够你重新摆造型的了，这样的功能完全解决了我们在自拍的时候点击快门的不方便。

奇酷手机相机在美颜模式下自带4种美颜功能，分别是美白、磨皮、瘦脸以及大眼，每个功能都有对应的调节进度条，调节不同功能后都能够进行实时预览，十分方便。

# 硬 件 #
360奇酷手机旗舰版搭载了高通骁龙808六核心处理器，集成两个ARM Cortex A57核心和四个ARM Cortex A53核心，GPU则是Adreno 418，内存配置为3GB RAM+16GB/32GB ROM，在这样的硬件配置下，奇酷旗舰版完全胜任了目前的主流应用和大型游戏。我们还是通过跑分软件来测试一下奇酷旗舰版的性能。

# 系 统 #
一部手机好用与否，除了手机外形、硬件配置外，最直接的决定因素莫过于操作系统的好坏了。众所周知360在软件界已经是业界老大地位，其安全应用也深入人心，那么奇酷手机搭载的360OS不可避免地就会系统的安全性上下功夫，在将近一个月的使用时间里，笔者深深地感到了360OS的便利性，许多细节之处当你真正体验到的时候你会情不自禁地发出“原来这功能都有”的感慨，下面就让我介绍一些我觉得360OS特别好用的细节吧。

右滑解锁、缓冲式下拉通知栏、底部上滑快捷开关，这是一些最基本的操作，也是用户平时做得最多的操作，360OS在这点上做得不错。


分类式的系统设置界面，并没有采用左右分页设计，而是从上至下，由于类目较多，查询相应的设置项的时候会比较繁琐，习惯之后也没有什么大碍。


系统安全性方面，你甚至发觉360OS在关心你的手机安全上做到了无处不在，体现在众多的安全应用和设置项，满足了你一切的安全设置需求，包括了云服务中心、手机防丢、手机找回、财产隔离系统、隐私空间、应用锁、安全中心以及应用冷藏室，以上这些应用汇总在一起，仿佛给手机筑起了一道最安全的围墙，不亏是以安全出名的360的做事风格。

# 快 充 #
奇酷手机支持涡轮闪冲技术，前提是采用自带的充电器进行充电，充电时系统会询问用户是否进行快速充电，当我们选择快速充电模式进行充电的时候，就会以9V 2.2A的输入功率进行充电，我们将奇酷旗舰版电量用空，然后从自动关机开始进行充电测试，期间WIFI和数据网络处于连接状态，无后台应用，实际测试从1%冲到100%用了1个小时20分钟，其实当冲到1个小时的时候，电量已经有91%，可见当电池电量达到90%后，充电速度已经开始降低，这么做也是确保电池的使用寿命。

# 总 结 #
好奇心能不能改变世界，这是一个需要长期验证的命题，可以肯定的是好奇心为我们带来了一款高质量的产品，那就是刚刚为大家做了开箱的360奇酷手机旗舰版，老周说他希望能用他的好奇心做出很酷的产品，而事实体验中这也是一款很酷的手机没错，甚至我们说他是黑马旗舰一点也不为过，1999元起的销售价格，具备了目前旗舰手机应当具备的大部分功能，6寸2.5d弧面玻璃 1080P屏幕、双1300万像素彩色+黑白夜视摄像头、背部指纹识别、金属机身、涡轮闪冲技术、自家360OS等等等等，如果再加入NFC和HIFI，那么这部奇酷旗舰版已经称得上完美了。但是具体口碑如何，还是需要市场的检验，就双11的小来销量来看，消费者似乎还是十分乐意接受这部酷酷的黑马旗舰的，你怎么看呢？
