反转无极限，娱乐办公二合一：VOYO VBook V3测评

3591 2


移动产品这几年逐渐吞噬PC产业，促进着PC 更多元化的发展，无论是体验，还是形态都发生了改变。现在大多数用户购买笔记本已经不是单纯的性能迭代了，他们会考虑多元化的需求，比如便携性、场景适应能力，或者是特殊功能功能。比如繁繁现在的ThinkPad 笔记本差不多已经用了5年左右，虽然对内部硬件进行了全面的升级，但2.27 KG的重量每天几乎让我累趴！对于我现在而言，还是主力在办公，所以对于便捷性十分重要，虽然之前有用过平板电脑，但没有实体键盘和鼠标，办公体验不够优秀。怎样才能将便捷和办公很好地结合起来呢？这次我选择了更加趋向笔记本的 Voyo VBook V3。



VOYO 是我们国内非常优秀的平板品牌，主打Android和Windows 双系统平板，去年推出了首款11.6英寸Win10平板 VOYO A1 Plus ，把性能与便携合二为一，真正做到办公娱乐功能的完美结合，而这次的Voyo VBook V3 更加强悍。Voyo VBook V3 采用13.3英寸1920*1080高清屏，170度广视角IPS硬屏，支持10点触控。性能方面搭配了最新Intel x5 z8300凌动处理器，4GB 内存，64GB硬盘，支持128G SD卡扩容，10000毫安内置锂聚合物电池。但Voyo VBook V3 合普通的平板不同，他和传统笔记本一样带有键盘底座，而且可以达到360°翻转。到底是一款怎样的产品呢？我们慢慢来看吧：



Voyo VBook V3 的包装非常的简洁，表面的彩格设计一眼可以看出是Win10平板，包装上印有的“VOYO TABLET PC”已经给产品进行了定位，打开包装盒内，除了主机外，还有电源适配器、说明书等，并没有特别之处，就不多说了。Voyo VBook V3 有香槟金、宝石蓝、宝石灰、时尚橙、钻石粉五色可选，考虑到之前都是用的银色合黑色，所以购买了颜色非常跳跃的时尚橙。




Voyo VBook V3 可以说就是超极本，太骚了！橘色太抢眼了，看起来很明亮、活泼系，显年轻，非常符合VOYO 一贯的风格。材质上选择了类肤质材料，触感不错摸起来很舒服，不过这种类肤质材料可能会随着用户使用而沾染污渍。机身正面中央有VOYO的LOGO，上面是摄像头，采用了前置200W，感觉用处并不大，相信你也不会拿个笔记本去拍照吧？不过还是给用户多一种选择。




Voyo VBook V3 的接口可以说相当全面，左侧分别是一个USB 2.0接口、3.5mm音频和SIM卡接口，音量键、电源键和键盘锁，几个按键的键帽不突出，几乎与侧面面板平齐，摸起来不容易。右侧则是一个USB3.0接口、电源接口、TF卡槽和MicroHDMI接口，可以说几乎无阉割，功能比较齐全。Voyo VBook V3更偏向于娱乐办公，HDMI接口 可以让我们在办公室放投影也可以在家里外接屏幕。




拿出Voyo VBook V3 给我的第一印象就是轻薄，厚度仅16mm，重量也只有1500g，无论是放在包里 ，还是手上都毫无压力，出门不会再有什么负担。比13寸的Macbook air 1350kg的重量差不了太多，虽然没有苹果轻薄 ，但相比传统笔记本那简直是“天使”。



Voyo VBook V3 采用13.3寸1920*1080高清屏，170度广视角IPS硬屏，可惜没有采用全贴合技术，而且屏幕边框我认为可以更窄一点，冲击力会更棒。实际观看屏幕艳丽通透，比较细腻，可视角度也很广，稍微有点漏光现象。而且不是单纯笔记本，他和平板电脑一样触控功能，支持10点触控，PC里面变平板。



Voyo VBook V3 底部设置了二个散热孔和四个防滑垫，底部材质和机身一样，采用了类似肌肤的轻便复合塑料材质，摸起来十分舒服，耐脏度还有待考验。




之前平板电脑都是搭配一个蓝牙键盘，但Surface 的出现，厂商们也纷纷采用磁吸键盘+docking接口，但一般这种键盘的口碑也并不是特别的好。Voyo VBook V3 键盘采用了常见的那巧克力键盘，键盘布局与普通笔记本电脑无异，拥有了出色的输入能力，但限于机器的整体尺寸，键盘的布局比较局促，键程适中，反馈比较绵软。而且配备了触摸板，它和标准的笔记本电脑键盘差别并不大，不过比较容易误触，有待提升。



大家再来看看屏幕和机身之间连接的金属的转轴，质感相当不错~



这款产品日常只是一个普通笔记本，但可以根据场景变换而采取更多模式，这得益于Voyo VBook V3 拥有360°旋转，可以变成笔记本模式、平板模式、站立模式和帐篷模式适应我们日常不同环境的使用。



视频，看图，适合使用站立模式，任意角度更加方便。



更加平稳的帐篷模式，涂鸦书画更加得心应手。



平板模式就不用多说了，触屏游戏、电子读物更加适合。值得一提的是还支持一键锁定功能，当我们需要转换成其他不需要用到键盘的模式时，为了避免用户触碰键盘造成误操作，可以锁定键盘，相当的很贴心的。




Voyo VBook V3 还配备了原笔迹手写笔，对于对于做设计的朋友，大大提高绘画体验，不过我觉得鼻尖硬了一点，感觉会弄坏屏幕似的。



Voyo VBook V3 采用了Intel x5 z8300凌动处理器，4GB 内存，64GB硬盘，之前平板受限于2G内存，所以软件并不能同时开得太多，否则系统会明显出现卡顿，这下问题解决了。另外内置了10000毫安锂聚合物电池，续航可达8小时。除此之外还预装微软正版Windows10 操作系统， 办公尽情发挥，虽然大型游戏还是比较吃力，不过Win10商城游戏还是妥妥的。虽然Win10 操作有多种选择，但常规的笔记本模式，键盘搭配鼠标，才是最配的。


总结：




Voyo VBook V3 的Intel X5+4GB+64GB的搭配，虽然不是顶级，但是性能上也表现不俗，日常的基本需求没有问题。超极本的形态提供了出色的办公生产力，360°旋转设计能够在不同场景下找到更多可能性，超长时间续航，很符合我们现在多元化的环境。当然也会有一些BUG和不完善的地方，相信日后厂商会优化。Voyo VBook V3 作为一款平板与超级本二合一便携式电脑，不管是文字编辑、视频、游戏都可以胜任，如果你是长期外出行走的商务人士，或者是喜欢多模式的年轻人，这款产品绝对适合你。
