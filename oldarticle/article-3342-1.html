不偏不倚不完美 联想乐檬K5 Note评测

7491 2
“真的好一点”来自联想对乐檬这个富有青春气息系列的产品定义，起初乐檬K3的发布正是想证明在千元档其实是有地位的——相对高规格的配置和低廉的售价，看似真要比普通的同级别产品好那么一点点。


当一个老品牌的新生系列诞生之时，往往为了稳重求胜而只推出一款机器。要么针对旗舰或是高端消费市场来吸引目光，要么就是在不影响本品牌的品牌形象前提下去争抢入门级市场的份额。在千元机市场的大门完全打开后，随着利益的权重化和大众消费观的转变，越来越多的厂商针对入门级消费市场相继推出中低端入门的机器。而乐檬K3的出现也正好用上面的理论进行解释，不过前段时间乐檬X3的登场反倒是让乐檬这个联想的子品牌又多了一层奇特的色彩。

作为乐檬系列最高端的型号，乐檬X3不管是工艺设计、性能用料、用户体验以及定价均在某种程度上可以说是一款中高端产品，而在上市一段时间之后也积累了相当不错的口碑。而如今手机市场更新换代之迅猛使得一款手机的的更新周期被至少于4个月，残酷的竞争让近乎所有的厂商不得不加快脚步去寻求最新的技术，而千元机受限于成本和定位，只能高举着性价比的大旗，肩负着争抢市场份额的重任。


很巧合的是，“性价比”这三个字从来都是千元机消费人群极为看重的。许多人认为“性价比”是一个俗且low的概念，毕竟老祖宗们那句“一分价钱一分货”在经过那么多年下来似乎确实是真理，这种所谓高配低价的产物从古至今都只是做到相对但从不绝对。因此，热衷于这类高配低价的消费群体相对来说对产品这样那样的小缺陷也是比其他消费群体们有着更多的宽容态度。虽然比不上那些四五千的苹果三星，但也对于够用党来说很多问题都能用“又不是不能用”这句金言来反驳，但若你喜欢时不时充值索大法和库克家的信仰，那就权当笔者啥都没说。


一款新的针对入门级市场的机器出现时，不管是配置还是千元级的价格，基本维持在一个固定的范畴，也就是保持跟竞品统级别的水平，对厂商而言不得不在配置和价格上妥协，定价基本都是以99或者999结尾，但依旧会拿些“黑科技”大肆宣扬。


千元机市场本身就是个比谁更廉价的定位需求集合，这类机器都有着一些相似性极高的特征——比如材质和功能模块的偷工减料以及某种难以在参数表上看出来的性能缺陷等。倘若从未长期使用这类机器，那这台乐檬K5 Note也谈不上有任何“惊喜”。正如服从家人命令前去相亲之前难免会幻想对方的身高和丑美，但一旦将期望提得很高，真正见面了后低于预期的失落感也就无可避免，当然咯，也许有些时候双方都很失望就是了。

外观工艺

是的，但从外观上看乐檬K5 Note确实给人一股平淡廉价的千元机的味道。但若是和其他国产品牌的做工和整体素质都还不错的千元机相比，乐檬K5 Note也不会逊色太多，特别是从塑料机身过渡至金属的处理。乐檬K5 Note后盖材质确实比较实在，银色的铝合金后壳也比相比起自己以及其他诸多千元机的土豪金版本显得更加低调有内涵。

不过美中不足的是受限于成本，乐檬K5 Note侧面边框与面板夹层的交界处处理得不够细腻，塑料边框仿造的CNC高光倒角也不够精细，衔接部位也比较粗糙，机器正面面板的左、右下角和边框之间甚至有些小缝隙。不过只要不是那种鸡蛋里挑骨头的处女座用户的话，这些都不是问题，毕竟乐檬K5 Note便宜的价格会让一切都还算是可以理解的。


不少人对现在广为流行的三段式金属机身设计可谓怨声载道，毕竟三段式的机身在材质上不统一，加上某些厂商的某些产品那不甚走心的衔接处理在许多时候能毁掉一个相当不错的外观设计。虽然千元机目标消费者相对来说不会那么吹毛求疵，但若是上下两端的塑料部分与金属的交界处有高低不平的现象，那么这款手机也难以给用户留下一个足够好的第一印象。乐檬K5 Note就是这样，三段平面高低有差，手感很糙。整体平淡的像一杯半沸腾的白开水，也注定了这款机器不会有像X3那样令人眼前一亮的能力。


右侧的物理按键就没什么好说了，手感保持了1000元上下机器的水平，按压手感也比较粗糙，加上偏长的键程以及一定程度的松动也瞬间暴露了乐檬K5 Note的低端定位，不过这都算还可以接受的了。


屏幕

乐檬K5 Note配备了一块5.5英寸全高清LCD屏幕，不论是尺寸还是分辨率都达到了千元“Note”们的主流水平。屏幕的素质还是参照以下实拍样张，对比机型为同价位的乐视1S，两者均调节至最高亮度，暗光下的屏幕色彩还原对比样张。(因相机实拍会有偏差，以下对比样张仅供参考，上方为乐檬K5 Note)








在显示部分多色彩较多的图片时，乐檬K5 Note的这块智能阳光屏饱和度是低于乐1S的。荷花那张的花瓣边缘尤为明显，乐1s的屏幕色彩饱和度、对比度都更高，整体风格趋近AMOLED。第二张的绿色草地对比中，乐檬K5Note的草地部分还原更接近图片原本风格。在默认色彩设置下，乐檬K5 Note因为屏幕饱和度偏低的原因，屏幕上橘红色的部分更接近于黄色。

不过还好，乐檬K5 Note系统设置之中可以就屏幕的色调、锐利度和饱和度进行调节，如此也算是给了对出厂设定不甚满意的消费者们一个根据自己喜好调节屏幕显示风格的办法。



要说乐檬K5 Note的屏幕最令人不满意的地方，那便是相对来说比较糟糕的可视角度表现了。虽然在正常垂直观看时乐檬K5 Note的屏幕亮度和色彩表现都算差强人意，但是在偏离45度左右之后乐檬K5 Note的屏幕亮度便会大幅度衰减。色彩也出现一定程度的偏红，对观感的影响相当明显，相比之下同价位的乐1s的屏幕可视角度就更好一些。




总的来说，乐檬K5 Note的屏幕发色风格是属于清淡型的。在正常观看时屏幕的色彩表现相比于同价位的乐1s显得更加清爽通透，色彩的过渡以及暗部的细节表现相比同价位其他手机也不显得逊色。虽然在和同价位竞争对手相比时，乐檬K5 Note的屏幕可视角度表现较为一般，但考虑到整机售价，乐檬K5 Note的 屏幕整体观感还是能够令人满意的。



系统体验

VIBE UI从诞生至今已过去了了三年，从青涩的1.0版本经过不停的打磨和完善，直到现在的成熟3.0，系统基于安卓底层的深度定制以及一些独家小功能都可以称得上是自家特色。不过相比起华为的EMUI、小米的MIUI和魅族的Flyme，联想VIBE UI的风格倾向并不算明显，并没有什么特别追求简约或是商务风格的痕迹。

比起VIBEUI 2.0的花枝招展，VIBE UI 3.0在原本已经功能颇为丰富的UI上还是做出了不少明显的改进和精简，以突出情感的共鸣这一联想“Vibration”理念的中心。举个例子来说，VIBE UI 2.0锁屏界面的“三大护法”在VIBE UI 3.0上被精简成只有上划启动相机和左滑切换壁纸的锁屏杂志界面。



VIBE UI 3.0图标风格跟随了时下最受欢迎的扁平化设计，默认主题各种图标下均为圆角矩形，布局看上去也相当和谐。默认的图标排列方式基本都是根据颜色相似度进行排列，以蓝色调为主的图标们放一起美观度也算不错。



VIBE UI 3.0的身上还是有着鲜明的特征的，系统中的特色功能里提供各类手势操作。诸如通过指纹替代相机快门键，单手快捷方式和智能场景等，虽然属于痒点功能，但却一定程度上提升了操作效率。


VIBE UI 3.0的文件夹操作在一众国产UI之中倒是颇为独特的，VIBE UI 3.0之中所有文件夹的切换无需返回桌面，简单地开启文件夹之后上下滑动即可在文件夹之间切换浏览。不过需要吐槽的是VIBE UI 3.0出厂内置的应用较多，这一点让人无法忍受，不过好在这些应用基本都能手动卸载，相对来说麻烦点罢了。



VIBE UI 3.0同时也加入了信息聚合模块，有别于TCL和乐视的系统那类的右滑到底进入最左侧的呼出方式，VIBE UI 3.0是通过下拉菜单再向右滑动调出的。该聚合信息栏显示内容自然是可以根据用户喜好进行定制，涵盖的可选内容分类共有15类，有科技、军事、体育和文化等等，这类功能也就是先根据用户定义再作出对应内容的调配显示。不过就笔者使用而言，目前VIBE UI 3.0提供的接入的合作资源较少，因此相对来说也有一定局限性。

“化简为繁，打破常规”是VIBE UI的设计理念，在VIBE UI 3.0之中的阶段虽是奔着极简主义设计，但VIBE UI 3.0的特色依旧不是那么深入人心。如果联想有意将VIBE UI打造成一个品牌以及旗下手机的特色，那么需要努力的方向其实还是蛮多的。


相机

今时今日，拍照功能已经成为检验一款手机好坏的标准之一。乐檬K5 Note的后置相机采用了一颗1300万像素的传感器，属于目前千元安卓阵营里主流的参数。但比较遗憾的是乐檬K5Note的相机虽然成像速度挺快，但操作体验却比较一般，对焦和测光的速度都偏慢。另外简单粗暴的白平衡、ISO调节等等设置选项的调出方式也令人感觉VIBE UI 3.0的相机界面完成度很低，整个界面就像安卓原生4.0时期那样缺乏特色。


在吐槽了乐檬K5 Note的相机界面之后，放几张实拍样张作参考。





由于拍摄当天是阴天，所以在某种程度上来说是不太利于乐檬K5 Note这类千元机产品拍照水平的发挥的。在实拍样张之中，乐檬K5 Note并没有对画面色彩饱和度进行额外的提升，发色非常接近真实情况。画面细节表现方面，乐檬K5 Note表现良好，对于锐化和降噪都没有进行过多的处理，虽然整体锐度和纯净度算不上同价位最佳，但至少画面比较自然。

在某些光线稍差的环境下，乐檬K5 Note的默认曝光比实际情况略高。而在逆光环境下乐檬K5 Note的摄像头宽容度会略显不足，需要用户手动开启HDR模式以平衡画面的明暗部分。






正如其他许多千元机一般，乐檬K5 Note的HDR模式是通过软件合成实现的。乐檬K5 Note开启HDR模式之后会显著提亮画面阴暗部分的曝光以保留更多细节，合成的效果基本令人满意。不过和同价位所有竞争对手一样，乐檬K5 Note在拍摄包含有移动物体的HDR照片时，合成的画面就会出现难以避免的“鬼影”现象。

受限于制造成本，千元机的夜间成像水平普遍一般，乐檬K5 Note自然也不例外。正如白天时的表现，乐檬K5 Note在自动模式下会明显提亮画面亮度，此时相对较高的感光度则会使得画面之中的噪点十分明显，在下方样张之中天空左侧更是出现了偏红迹象。虽然乐檬K5 Note可以通过压低曝光以换取更好的画面纯净度和细节表现， 但相对更低的曝光也会使得画面整体的观感并不讨人喜欢。




指纹识别

也许是这段时间指纹机发布的太过频繁的原因，乐檬K5 Note配备的指纹识别其实并没有给笔者多大的惊喜，而在诸多国产手机纷纷配备指纹识别模块的大环境下，识别速度和应用场景也就成了各大厂家大做文章的地方。联想官方称乐檬K5 Note的指纹识别速度最快至0.3s，因此笔者对乐檬K5 Note上的这块深深下陷的指纹识别还是蛮期待的。乐檬K5 Note的指纹解锁硬件采用的是来自FPC的1022触摸式传感器，感应面积更小但是精确度还是很可观。

在使用指纹解锁手机时，VIBE UI 3.0有意省略了过场动画，从而使得解锁过程在主观感受上十分流畅迅速。另外乐檬K5 Note虽然支持最多五组指纹，但却不像其他许多智能手机一般支持相同指纹的反复采集。而除了单纯的解锁屏幕之外，乐檬K5 Note的指纹识别模块还能够完成支付和拍照等日常常用操作。



续航&充电

比起上一代，乐檬K5 Note进步最为明显的地方在于将电池容量提升到了3500mAh，并且做到金属一体性，电池自然不能像K3 Note那样可拆卸可更换，于是多出了这500mAh容量就给了乐檬的老用户一剂定心丸：可以更愉快的使用 了。那乐檬K5 Note的续航表现到底会不会令人满意呢，先下个结论：乐檬K5 Note移动版能坚持普通用户1天半的使用时间，属于绝对够用的那种。

在千元机定位中，乐檬K5 Note的电池容量属于同级别较高的水准，普遍是在多出那500mAh带来的安心。实测中，乐檬K5 Note的续航表现颇为满意，在进行了如下的日常使用操作后，乐檬K5 Note的剩余电量可观，达到80%，图表中可以看出，曲线最陡的区间为游戏测试，耗电最为明显，要知道，游戏环节是最考验手机性能和掉电速度表现之一，K5 Note的续航表现属于所测千元机型中的中上等水平。


在这个快速充电技术盛行的时代，越来越多的手机配备了该功能，甚至连苹果都推出了快充适配器预拓宽自家的产业线。而在消费者眼里，没有快充功能的智能手机，往往都是抱以嗤之以鼻的态度对待。2015年的后半段，千元机大军也已享受到了快充时代带来的高质高效，人们的需求也随之变高。虽是如此，乐檬K5 Note的快充却普普通通，5V/2A的参数也是不痛不痒。测试中，由于乐檬K5 Note电量在2%时就已自动关机，因此测试充入电量即从2%开始，从图表中看出，乐檬K5 Note所搭载的低电压低电流式快充并优势不明显，花了足足150分钟才充满，略显尴尬，不过再看看明显的续航优势，1099的价格还是别要自行车了，适合就够了。


性能

乐檬为K5 Note配备了联发科Helio P10处理器。在联发科的官方参数之中，Helio P10的CPU部分采用了8颗最大主频达到2GHz的Cortex-A53核心，搭配以700MHz的Mali-T860MP2图形处理器。但是不知是否是为了控制整体的功耗和发热，联想决定将乐檬K5 Note身上这枚Helio P10的CPU最大主频限制在1.8GHz。

不过还好，实际体验中Helio P10体验并不会给人以拖泥带水的感觉，针对轻便型游戏的运行和网页的浏览基本不存在什么问题，但是对于那些较真的的性能党来说乐檬K5 Note所采用的2GB RAM终究是显得相当尴尬。虽然日常的操作流畅度颇有保障，但是2GB RAM还是注定了乐檬K5 Note的后台运行应用会相对来说很频繁地被系统杀掉以保证前台应用拥有充足的运行性能，所以总的来说乐檬K5 Note的性能水平还是仅适合那类不爱折腾的用户们。




乐檬K5 Note是一款合格的千元机，但平庸的配置加上不痛不痒的创新并不能让他成为一盘非常令人食指大动的油焖大虾。在小米、魅族、乐视等等更加“互联网”的品牌不断地推出各种大菜级别的千元机来互相搏杀的当下，乐檬K5 Note更像是自助餐柜台上红绿点缀的辣椒炒肉，也许它能够博得食客们驻足一视，但论存在感还是口味，还是难以和主流们抗衡。
