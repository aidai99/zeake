全“心”进化 nubia UI V3.9.6体验

19349 6
不管以任何标准来看，一台智能手机的用户界面在很大程度上决定了用户使用这台智能手机时的体验。一个好的UI除了功能强大之外更加是需要好看好用，单纯的功能堆砌并不一定能够带来最好的使用感受。


2016年4月19日，努比亚在北京召开发布会正式发布了旗下全新的小屏幕旗舰智能手机nubia Z11 mini。Z11 mini上面除了其相当出色的硬件配置以及拍照能力之外，其所搭载的全新nubia UI V3.9.6版本也是相当引人注目。除了诸多全新功能之外，其更新的视觉设计以及操作流畅性和前几代nubia UI相比更是有了飞跃性的进步。


以最新发布的nubia Z11 mini为例子，其所运行的是基于Android 5.1.1操作系统的nubia UI V3.9.6。在全新一版本的nubia UI之中，用户首先能够感受到的是诸多视觉上的修改。该版本的nubia UI之中采用了当下流行的扁平化设计元素，但是除了单纯地扁平化之外，nubia UI依旧拥有拟物化的设计风格。而对于那些追求个性化的用户们来说，nubia UI所提供的全局主题功能能够更进一步地让nubia UI贴近自己的喜好以及审美。


为了保证手机的续航能力以及日常操作的流畅度，nubia UI直接内置了手机管家应用。该应用将手机内存清理加速、应用权限管理、流量监控、骚扰拦截、手机找回以及电量节省六大功能做成了一个简单清晰的入口，在此应用之中可以手动选择是否允许某一特定应用在后台运行从而降低手机运行耗电。如此也是Z11 mini在配备相对其他Android智能手机来说更小容量的电池的前提下获得良好续航能力的原因。


在Z11 mini发布会上，努比亚表示考虑到现在非常多的用户都同时拥有多个社交网络账号，所以在全新版本的nubia UI之中加入了“应用分身”功能。该功能支持微信、QQ等等应用的“一机双号”，在各自登录之后两个账号之间互不干扰，加上努比亚诸多智能手机提供的双卡双待功能，更加方便用户平衡自己工作以及私人联系。


在全新的nubia UI V3.9.6之中，努比亚还加入了极为实用的全局分屏显示功能。比起竞品厂商传统的需要应用本身支持的分屏，nubia UI V3.9.6的分屏功能可以轻松实现任意应用的分屏显示。在开启分屏功能之后，屏幕被分为上下两半，将当前正在前台运行的任意应用置于其中一半之后再另一半窗口之中可以开启安装在手机上的任意程序，甚至还可以直接显示桌面图标。这使得努比亚Z11 mini的分屏功能拥有真正的可用性，无形之中也极大地提升了整个手机的智能度。


另外，在启用分屏功能之后，用户还可以根据自身需要选择对分屏窗口进行填充或是保持原有应用分辨率比例进行等比缩放。如此甚至值得nubia ui V3.9.6拥有能够和Windows操作系统的PC相媲美的窗口管理便利度，在诸多使用场景之中也免去了用户在应用之间来回切换的烦恼。


另外在nubia UI V3.9.6之中，努比亚也加入了部分早前于努比亚Z9上首创的边框交互功能。虽然比起Z9多种多样的操作方式，Z11 mini所运行的nubia UI V3.9.6在功能性上仅提供左右屏幕边缘滑动切换后台应用或是清理内存等最实用的功能，但这并不妨碍边缘手势功能在日常使用之中为用户提供便利。只要经过一些设置之后，用户便可以在自身顺手的左右屏幕边缘上下滑动以切换运行的应用程序、清理内存、启动特定应用等等功能，而屏幕两边的边缘交互则依旧提供实用可靠的屏幕亮度调整以及音量调节。并且为了更加直观地展示用户目前的操作，nubia UI V3.9.6在用户使用边缘触控手势触发某一功能的时候会在屏幕对应位置显示一条蓝色的色带对用户进行提示，使得这一功能相比之前的努比亚产品显得更加成熟可靠。


作为nubia UI V3.9.6上面最大的亮点，超级截图功能是不可不提的。努比亚的超级截图功能除了通过桌面上的图标开启之外，还可以通过长按电源键+音量键或是在亮屏时长按指纹识别模块开启，使用起来颇为方便快捷。


在超级截图功能之中，nubia UI V3.9.6的屏幕录制功能可以直接录制一段时间的屏幕显示内容生成视频，在手机上直接编辑并分享。并且更重要的是，在录制屏幕内容的时候用户甚至可以加入自己的讲解录音。如此不论是录制游戏游玩内容，或是录制某一应用程序的使用教程都非常方便。


nubia UI V3.9.6的自由截图功能也非常有特色。在截取屏幕内容之后，用户可以对屏幕进行矩形、椭圆，心形和自由形状等任意形式的截取，截图后可选择四种画笔颜色加以批注和描绘，也可进行马赛克处理，完全无需另外安装第三方软件。整体实用性、趣味性和易用性堪称业界最佳。


至于长截图功能则是努比亚针对用户对于网页、微信聊天记录、微博和朋友圈等等较长内容的截取所开发的实用功能。在按下按键之后手机会对屏幕内容进行自动滚动截取，用户可以在自己需要的时候停止。如此跨越屏幕界限的截图功能自然也免去了用户频繁地翻页和单屏截图的烦恼，不论是用于分享还是日后回看都会拥有更加出色的用户体验。


最后则是nubia UI V3.9.6的单手操作模式了，在开启该功能之后长按屏幕下方左边或是右边的触控按钮便可以开启/退出。该模式在第一次开启时会提示用户在屏幕上划出单手可以覆盖的屏幕范围并且为每个用户量身定制适当的单手操作界面大小。当然了，在制定完成之后也可以对但受窗口大小进行手动调节。


一个好的用户界面可以变成一个手机品牌的核心竞争力，nubia UI日积月累的更新体现的更多是努比亚对于用户的关怀。在跳出了简单堆砌功能的怪圈之后，nubia UI变得优雅、成熟、稳重、高效并且流畅，经过精心锤炼所打造出来的nubia UI V3.9.6不再是努比亚智能手机的一个附属品，其更大意义上在于汇集了努比亚品牌的精神，作为努比亚品牌在面对消费者的一个优良形象。努比亚最新发布的Z11 mini在搭载了这套UI之后也拥有相当强力的竞争力，在曾经被视为国产UI标杆的MIUI愈发臃肿，广告无孔不入的当下，能够有nubia UI这样一套功能强大用着舒心的用户界面，也算是一件幸事。
