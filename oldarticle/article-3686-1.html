追求极致的偏执狂 – Moto X极

18262 8
阅前须知
本文主角是一款追求极致的手机，且文章带有作者强烈的个人色彩，不喜慎入！
阅读过程中若出现明显不适，请及时点击关闭！


1973年，Moto实验室的极客马丁·库帕率领他的团队，发明了世界上第一部手机，从此，人类进入了无线通信时代。而凭借着纯粹的极客风范和创新精神，数十年来，一代又一代的Moto人发明了数不清的黑科技，创造出了无数经典产品。


从功能机时期首次使用航空铝合金材质最薄最酷的翻盖手机RAZR V3、首款圆形蓝宝石屏幕“珠光宝气”的AURA、魔幻音乐界面的E8、透明翻盖的明系列，再到Android时期外观独树一帜的后空翻、侧空翻、金属感十足的里程碑、接入笔记本扩展坞即可变身电脑以及首款配备指纹识别功能的智能手机Atrix、颠覆了智能手机制造模式和用户购买模式的Moto Maker，全球第一款的圆形智能手表、将智能手表做成时尚品的Moto 360……摩托罗拉，为移动通讯和智能终端做出了太多太多的贡献，将手机制造技术与艺术一次又一次带上新的层面，这绝对是一个当你深入了解后，肃然起敬的企业！


在2015年的年底，摩托罗拉推出了全球首款配备了ShatterShield极御式防碎屏幕的手机 -Moto X Force（美版名为Droid Turbo 2，国行版名为Moto X 极），该机的特色就是配置了一块号称“摔不烂”的显示屏。


现如今，智能手机的屏幕越来越大，屏幕也成为了整机中最脆弱的部分，一个不小心手机着地屏幕就碎了。不少智能手机用户都有遭遇过碎屏的经历，而其中大部分用户由于维修成本过高或者对手机的依赖太强，还在坚持使用屏幕出现裂纹的手机。而另一方面，大部分手机厂商也都选择了从产品本身回避这个难题，通过给手机用户提供附加的碎屏险来应对碎屏这件事。

而摩托罗拉这个偏执狂，却本着骨血里的那种极客精神，竟然花了将近三年的时间，通过ShatterShield极御式防碎技术真正的解决了碎屏这个难题。80多年的历史给摩托罗拉带来的，并不是老气横秋，而是一种不分岁月的酷，那种绅士与骑士交融的酷。


据介绍，Moto X 极采用的ShatterShield极御技术显示屏由五层创新保护层组成，充分吸收手机跌落时所受到的冲击力，保证屏幕不会破碎或破裂。而最外层的极御防碎层配备了专有硬罩，可以有效防止凹痕与磨损，是整块屏的“金钟罩”。但这也是最受考验的一层，经过长时间使用后，屏幕表面还是比较容易留下细小划痕。或者手机遭遇重大“事故”后也可能给屏幕带来明显的伤痕，这些对于强迫症患者来说自然是难以忍受的。


好在Moto X 极的最外层极御防碎保护屏可以轻松更换，让你的手机屏幕始终完好如新。Moto官网也提供极御防碎保护屏单独出售，售价199元。可以自行更换最外屏，让手机重获新生。但花费199元来替换最外层保护屏的成本确实也并不算低，是否需要为了自己的强迫症来为之买单，还是自行权衡吧！


不过呢，所谓的“摔不烂”仅是针对日常生活中的误摔，譬如从口袋、手中或桌上摔落等情况。当然有外国朋友对其进行高空275米的跌落测试，手机屏幕依然没有损坏。但为了您的个人利益，还是请勿尝试锤子砸、刀子刮或重物碾压等过于暴力的测试。毕竟Moto X极的“耐摔”与车辆的“经撞”一样都只是相对的概念，无论车身结构多么先进、钢材等级多么高强、电子辅助系统如何智能齐全，都必须建立在文明安全驾驶的基础之上。相信也不会有消费者会为了证明自己的车“经撞”，而刻意将自己置入危险的境地。


说到这，不禁要对摩托罗拉的偏执程度发表一丝发自心底的感叹。防碎技术确实在某些意义上解决了许多用户的痛点，让手机更耐用。在曾经的功能机时代，手机可以砸核桃甚至是可以当做“防身利器”的传闻也一直不绝于耳，可见产品的耐用性在消费者心里的位置是非常之高的。但到了如今的智能机时代，智能、便捷、易用渐渐地已经取代了耐用的位置，在消费者心里变得更加重要，手机甚至已经变成了一种快消品。在这样的大环境下，摩托罗拉依然花了近三年的时间，将手机的耐用性又重新拉回大家的视线中。抛开创新与核心竞争力不说，就摩托罗拉这种不跟随的偏执与坚持，是值得认可与钦佩的。


防碎屏，是否能够成为Moto X 极的一个强有力的卖点，暂时很难回答，但不可否认的是其绝对会成为Moto X 极，甚至是当下整个手机行业的一个亮点。毕竟这并不是一款主打性价比的机器，不低的售价也许会让不少用户望而却步。但可以肯定的是假如Moto X 极的售价更加亲民一些，或者防碎屏技术逐渐普及至更多产品线后，一定会有更多消费者愿意为之买单。


接下去来详细说说Moto X 极这台手机。

Moto X 极的包装盒采用了摩托罗拉近年来延续的彩色几何拼块的设计。打开包装，除去手机外，里面附有耳机一副，Turbo Charger充电线缆一套（充电器与数据线不可分离）、用户指南及保修手册等若干。也许是出于为成本考虑，配件中并没有另外赠送数据线，不过对于现代人来说，应该是不会缺数据线的，与大部分手机、相机的数据线都是通用的。



拿着Moto X 极的第一瞬间，我的脑海中冒出了这样四个词：“力量”与“优雅”，“机械”与“情感”。这些看似冲突的词汇，融汇到这台手机之中，散发出一种与众不同的气质。Moto X 极属于设计优美但个性强烈的一款机器，这样的风格显然非常容易导致一个众口难调的问题。Moto X 极的外观带有非常典型的摩托罗拉风格 - 刚毅、硬朗、粗矿，同样支持moto maker的定制融入了时尚、个性的元素。尤其是机身背面，搭配不同的材质，更让整机精致感与品质感向上迈了一大步。


摩托罗拉近年来多款机器的后盖都是最具话题性和吸引力的部分了，Moto X极也同样提供了多种材质可供选择。包括奢华真皮、防弹尼龙及数十种颜色的时尚硅胶材质。


皮质后盖来自著名的皮革供应商：Horween，这可是众多奢侈品皮具的供应商，例如意大利的Prada、英国的John Lobb、以及美国的Allen Edminds都采用Horween的皮革。皮革的触感亲肤细腻有弹性，冬季甚至会有温暖的错觉，与侧边金属的冷冽形成鲜明的对比。虽说皮质版本的手感柔软光滑而细腻，且质感更易体现出手机的档次，但相对也比较娇贵，需要小心使用，并注意平时的爱护与保养。官方提供荔枝纹的干邑色皮、本色皮、黑色皮三种颜色可选，注意，在Moto Maker只有皮质后盖支持免费刻字。


由于尼龙面料具有质感轻巧，强度高，耐用，防皱不显旧等特点，让其广泛运用于各种包包、衣服、以及时尚单品中。近年来不少数码产品中也加入了尼龙材质，在新元素的帮衬下，不少数码产品也越发突显出利落、简约的时尚色彩。Moto X极尼龙后盖细密编织的纹理，带来了凹凸有致的独特触感。扎实的手感，更是给我前所未有的踏实。靠近可以明显观察到尼龙的截面和纵向结构以及丝状的原纤维组织，乍一眼看非常像穿有护甲的战士，总之防弹尼龙后盖是最受本人喜爱的一个版本。



多彩的硅胶材质后盖也毫不逊色，手感温润，触感亲肤，且颜色的选择更多，菱形的细纹路所产生的摩擦力也使得握持更加踏实。唯一的缺点就是格子的纹路中容易藏污纳垢，当你靠近细看时，纹路中的小灰尘总会让你产生手机很脏的错觉。




Logo、双色温LED闪光灯以及摄像头位于背面上部的金属长条中。将机身横放时，Logo和摄像头的两个圆圈搭配金属长条的边框，乍看还挺像个传送带，也有点像磁带的供带轮。金属长条以及环绕相机镜头的金属镶边的颜色也都支持定制，不同的颜色与材质的搭配，更增加了时尚感，彰显个性。


机身中框为金属材质，Moto X 极并没有延续Moto X系列贴合手掌握持弧度的曲线形设计，别具一格的拿破仑帽额头也不复存在。所以相比Moto X系列之前的机器，Moto X 极的整体造型缺少了一丝优美的灵动感，但由机身中间向两边延伸依然采用了一个斜切的设计，同样也使得手机与手掌更加的贴合，增添了一份踏实感，整机重量适中，再加上不同后盖材质的纹路，不会有手机会随时滑落的错觉。


按键布局也与以往相同，左侧干净整洁，没有任何开口及按键；右侧由上至下为电源键和音量键。电源键表面的细纹处理与音量键有所区别，可以增加盲操时的简易度；机身上方有3.5mm耳机孔及卡槽，机身下方为USB接口。


Moto X 极的面板与Moto X系列其它机器屏幕上下采用对称的双金属条设计不同，Moto X 极仅保留了听筒处凸起的金属条，采用了梯形设计，两边各有一个切角。也许是为了和后盖的切角形成某种意义上的前后呼应吧。而屏幕下方变为了两条横向开孔，但仅右边的开孔是扬声器，虽然放弃了双扬声器的设计，但外放音质表现整体还是不错的。



与此同时，Moto X 极的面板也是整机争议最多的地方。由于聚集了摄像头、闪光灯、四个红外线感应器、距离感应器和光线感应器，还有分别位于屏幕上方支持自定义颜色的听筒，以及下方左右对称的麦克风及扬声器开孔，整个前面板基本上形成了一个典型的麻子脸。很多“处女座情结”的朋友表示难以忍受这些密密麻麻的小黑孔，尤其是白色面板显得尤其明显。而黑色面板能起到一定的隐藏效果，让整体的视觉效果好一些。



丑和美，既是一个客观的存在，是审美客体的自然属性，也是一种主观感受，是审美主体对于自身本质力量的确证。对于丑和美的审美评价，也就和主体的审美态度相适应。在贾平凹的《丑石》中所提供的语境中，曾说道：“丑的极致，便是美的极致”。而这台手机在外观方面，也许也正好印证了其命名“Moto X 极”中的“极”字。虽说这些开孔在某种程度影响了面板的整体性，但也正因为这些开孔的感应器，重新定义了智能手机的“智能”二字。随时感知你的动作、声音、所处场景等，并根据你的习惯做出相应的反应，这一系列从生活细节出发提高效率的差异化功能，让手机变成了一个有生命，有思维的个人助理。得益于先进的摩托罗拉移动计算系统（Motorola Mobile Computing System），为我们带来不同的响应体验。该系统包含经过软件优化的高通骁龙810处理器、自然语言处理器以及语境计算处理器，使得智能的Moto显示（Moto Display）、方便的Moto操作（Moto Actions）、贴心的Moto助手（Moto Assist）以及随时待命的Moto语音（Moto Voice）得以应用。经过时间的验证，这几大功能都备受好评且极易使用户产生依赖性，算是摩托罗拉体现其品牌差异化的杀手锏。



Moto操作借助于手机前面板的红外线系统（三个发射器和一个接收器组成），检测用户的手在上方十厘米以内挥过的动作从而实现来电静音和关闭闹钟等操作。或配合Moto显示还可使其主动显示时间以及未读通知,极其方便。当你从口袋、桌面、包包将手机拿出时也能得到相同的效果，无需按下电源键，手机随时等待用户解锁或者在屏幕向上划动进入相应程序。另外握住手机晃动手腕两次即可开启相机，轻甩两次开启手电筒功能的自然手势交互也让Moto X 极显得别具一格。这一系列功能的优势就在于省去用户繁琐的操作步骤，显著提升信息处理效率。


Moto Voice语音唤醒功能。即使手机在息屏状态，依然可以使用口令“Ok, moto x”唤醒手机，并可以使用中文通过语音进行很多相关操作，向手机提出问题或是发出相关操作指令，无论网络搜索、定位导航、发送短息，只需使用相关口令后下达指令即可。


Moto助手是一个情境式学习的引擎，可以自动激活并调整手机的诸多功能以适应用户的需求。例如开会、驾驶或晚间入睡不想被打扰时，Moto X 极可自动开启静音通知并自动回复信息或读取短信、告知呼叫人等功能。


系统方面，Moto X 极依然搭载了接近原生的Android 5.1系统，内置应用非常少，仅有新浪微博、虾米音乐、高德地图、讯飞输入法、茄子快传及天翼对讲6个，且全部支持自行卸载，还是很良心的。不过我想单独提一下天翼对讲，由于Moto X 极在国内上市时与中国电信合作，推出了一系列购机话费补贴的活动，顺理成章的也就内置了电信的这个应用。功能方面看官方介绍貌似还不错，由于本人电信卡并未开通该项功能所以无法体验。但这个应用的图标以及界面着实太粗糙，感觉一下子被拉回了Android 2.X的那个时代，且一旦开启便会强占通知栏无法退出。虽说这个应用并不支持第三方商店单独下载，预装在Moto X 极中显得还有些专享的优越性，但如此粗糙的一个应用内置于一台定位旗舰的手机中，还是显得有点格格不入的。

MotoX 极系统界面

天翼对讲应用界面

Moto X 极自在国内市场上市到本文完稿，已经有5个余月了，系统依然还是停留在Android 5.1版本，根据网络上不少用户的反馈，大家对于国行版的升级速度普遍是不大满意的。毕竟Moto回归大陆市场后，就以流畅的原生系统以及更及时的系统升级作为卖点。还是希望联想及Moto能够尽快为旗下产品推出系统更新，在这块可千万别做一位偏执狂。希望几年前多少人抱怨摩托罗拉升级慢、声称“再买Moto剁手”的现象不要再重演。


相信Moto X 极所搭载的骁龙810处理器也一定是个重要关注点。鉴于之前许多搭载该处理器的旗舰机的表现，让人们对810望而却步。不过就我个人的对比与体验，虽说发热还是存在，但Moto X 极的流畅度、续航和散热控制在搭载810处理器的机器中做得算是最棒的，有非常明显的改进。毕竟Moto X 极推出比较晚，算是810批次的末班车了吧，也给了比较充裕的时间去调校及改善这颗火热的芯。所以810这个梗基本可以在Moto X 极身上忽略。


另外，3G+64G的内存组合并支持最高2TB的Micro SD卡内存扩展，绚丽的2K超高清5.4寸显示屏、内外双层防水纳米涂层、续航时间长达30小时的3760 mAh电池、支持涡轮快速充电和Qi、PMA标准无线充电、支持 Smart boos的前置扬声器、支持盲插4G+2G (GSM/CDMA)的Nano SIM双卡双待、蓝牙4.1LE、双LED色彩补偿CCT闪光灯、支持相位自动对焦的2.0光圈2,100万像素摄像头，这些参数也足以奠定Moto X 极的旗舰地位。



相信不少人还记得MotoX Style采用的三卡一体式的卡槽，即双NanoSIM卡及Micro SD卡可以同时插入卡槽。但遗憾的是Moto X 极并未将这种设计延续下来，而是采用了国内比较普遍的“与或卡托”，即支持同时插入两张NanoSIM卡，或者单张Nano SIM卡搭配一张Micro SD卡。Moto X 极支持双卡双待全网通，且支持电信新六模，电信卡可选择作为主卡或者副卡，但不支持同时插入两张电信卡。



Moto X 极还配备了5 个麦克风，支持新一代的Moto丽音降噪技术，该技术使用五个麦克风和多种数字信号算法，从对方环境噪音中过滤出语音，同时根据本地环境噪声，对通话音量加以调整，即使在嘈杂的环境下依然可以让通话更轻松自如。在通话质量已经基本被忽视的情况下，摩托罗拉依然不忘手机通话的本质，让MotoX 极的通话质量又一次遥遥领先于其它机器。

介绍到此，对Moto X 极应该已经有了一个比较全面的介绍，如果要做一个总结，我个人认为Moto X 极是一台庄重、实用、不显浮华，又非常具有辨识度和特色的手机。没有指纹识别，没有一体化的全金属材质、没有三段式的后盖设计、没有凸起的摄像头，但Moto X 极有防碎屏幕、支持多种不同的材质和颜色的自由定制、接近原生的Android系统、支持新一代的Moto丽音降噪技术等等，没有跟风，没有抄袭，浑身充满着令人刺激的科技创新，在以市场本地需求为导向的低调枯燥的产品中显得那么的与众不同。

“不走两极，勾势恰作”？那不是摩托罗拉。
