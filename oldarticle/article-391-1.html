不甘平凡，雨晨大婶分享酷派大神F2

1276 1

    2014年作为中国4G的元年各手机厂家借助4G之风大推旗舰机，3G手机仿佛在一夜之间已经悄然暗淡。看过了以各种“噱头”为主打动辄3000-4000元的旗舰产品冷静下来思考下：我们是否真的需要那些功能和配置？我们是否在花着几千元的代价却用着几百元的功能？在千元机市场上有没有一款实实在在的能够满足用户各方面的需求。这段时间上手一台酷派大神F2正好处于千元价位，下面就简单聊一下使用体验。

    大神F2上手给人第一印象简洁轻薄，前面板没有多余的装饰整个造型规规矩矩，这一点是我比较喜欢的，很多厂商不知为了彰显自己的品牌还是想让更多的人记住会在手机顶部印上自己的LOGO，漂亮么？不觉得。非但对于自身品牌没有什么提升反而会显得有些廉价。

    手机前面板整体为黑色基调，配合全贴合的屏幕整体统一性十分的强。当黑色调统一时，我们很容易想到一个词“息屏美学”，大神F2在这方面做的还是不错的，从图中可以明显的看出来只有在泛光的特殊角度才能看到内屏，而正常情况下阴影部分是看不到内屏和边框的分区界限的。

    大神F2采用了一颗500万像素的前置摄像头，屏幕上方的听筒金属网罩部分是与整体前面板齐平的，如果严谨的说是略微高出整体屏幕面板一点点的高度，这样能够在长时间使用中不至于积累灰尘。同时细心的观察屏幕边缘的边框也会发现屏幕边缘的边框也是稍高出屏幕一点的高度，当手机倒扣在桌面上时，桌面和屏幕不会直接接触从而能够在一定情况下减少和减轻了屏幕和桌面摩擦造成的屏幕划痕。

    这一点的设计可能不太容易发现，当对比了几部手机后确实发现边框整体高于屏幕要比别的手机明显一些。

    屏幕下部的三个虚拟触摸按键设计的比较有意思，中间的home键为浅灰色圆形设计。相对于常见的安卓小房子home键更简洁漂亮。其实不是真正的圆形，而是边角比较大的矩形。功能键和返回键隐藏与左右两侧，并且有背灯的设计可以选择打开背灯或者是关闭。当关闭时整个触摸区域就仅剩这一个“圆形”整体感相当的强，当打开背灯时左右两侧的功能键和返回键会亮起。同时这个背灯支持时段自定义，晚上打开方便操作，白天关闭做到整体性的美观。

    我们再看大神F2的背部，背部设计应该说十分规整。左上部为一颗1300万像素的摄像头并配以单科LED闪光灯；中部为一颗降噪MIC；下部为酷派的LOGO和扬声器的开孔。整体设计还是以简约为主，其实这里值得提的一点是背部后壳的处理。千元价位的手机后壳基本都为塑料材质，虽然材质并不出奇但是处理的方式确是多种多样。如红米的类磨砂处理、荣耀3C 4G版的仿皮质纹理处理、中兴红牛V5的光面处理等等。酷派大神F2在后壳的处理方面设计了一个细小简洁的纹路。能够反馈给手类似于磨砂的质感但又比磨砂深刻细腻，这一点还是十分值得肯定的。

    酷派大神F2的厚度为9.15mm，这个厚度在现在超薄设计满天飞的现在并不出奇，甚至可以说没有什么优势。但是我们回归实际体验，过于薄的手机确实能够在视觉方面给人以惊艳的视觉满足感。但是手机厚度设计是一个双刃剑，首先过于薄的设计不会有一个合理的弧度从而得不到一个好握感，说白了就像是如同拿着一块玻璃板一样咯手。其次，手机的厚度也会影响电池的容量。而大神F2则是做了一个综合性的平衡。

    按键方面音量键设计在了手机的左侧，感觉位置有点稍微偏上。电源键则放在了右侧，按键采用了金属质感的处理，并且在按键表面做了纹路的处理提升按键手感的同时也提升了档次。同时从这个角度来看我们也能看出摄像头是平与背部的并不突出，现在这个时代连iphone6的摄像头都变成了火疖子，想找一个平的摄像头手机真的不容易。

    在手感方面由于F2的R角为比较大的椭圆设计，同时在背部后壳也做了相应的弧度处理以及9mm的厚度。综合这些方面来说大神F2的手感在5.5寸这个尺寸的机型中还是相当不错的。

    OK，我们再谈一个不容易注意到的细节。大神F2从正面看是黑色的屏幕面板及边框为一个整体。在这个整体之外是包围着一圈白边的，这个白边就是后壳。我们仔细看会发现这个后壳是要高于黑色部分。所以这个细节第一能够当做一个白色的装饰边框，不至于一体的黑色过于单调。第二，由于这个后壳是包裹于整体的前面板，所以当手机跌落时，后壳会着地起到一个缓冲的作用进而降低碎屏率。

    显示方面大神F2采用了一块5.5寸分辨率为1280*720分辨率的显示屏，在现在1080P分辨率随处可见2K屏幕也不新奇的今天来看这块屏幕的分辨率并不算高，好在这块屏幕的色彩和亮度均有着较不错的表现，即使在户外由于屏幕亮度较高，所以屏幕内容也清晰可见。

    大神F2配备了双SIM卡槽，同时也支持存储卡扩展，在这方面还是比较厚道的。标配了一块2500mAh的电池，在续航方面轻度使用两天重度使用一天是没有问题的。

    简单聊完外观工艺以及设计之后我们再看大神F2的系统，硬件是基础平台而真正影响用户实际交互操作的还是系统，所以一部手机易用与否系统也有着不可忽视的影响。

     系统方面大神F2采用了自家的全新的Cool UI 5.7，在锁屏界面下方设计了电话、短信、相机三个快捷图标，滑动可以直接进入功能。进入桌面取消了二级应用页面，采用四列图标拍了所有应用都放在桌面，让整个系统更为简洁操作体验也是十分流畅。图标采用了椭圆扁平风格，在图标的底部淡色投影使的图标有种浮于壁纸之上的效果。

    功能设置界面采用了浅色基调并加以灰色暗线区分功能分区，简单清晰明了。桌面文件夹打开后整体界面会有变成朦胧的毛玻璃感觉，下滑任务栏也是如此。同时状态栏也会变成与之相对的色彩，类似于沉浸式体验。

    由于安卓系统的同质化，所以Cool UI 5.7和常见的手机系统并无本质区别，亮点在于简洁的设计、流畅的操作以及系统的纯净。值得提的一点是Cool UI 5.7在设置里单独设置了一个“大字体关怀模式”，其实说白了就是一个“老人模式”。开启这个模式之后手机会重启进入大字体关怀模式，这时的桌面也会变成简洁的大图标，同时设置界面里的字体也会加大。这一功能设计无疑是为视力不佳的老年人而设计。

    谈到这里必须要聊一下“大字体关怀模式”，它并不是一个主题，而是一个模式。之前也见过一些其他品牌的机型或者主题软件也会有这种类似的模式，但是只不过是一个主题而已，应用之后仅会在桌面出现大图标，而当进入某个功能之后如拨号。还是会出现普通模式下的拨号界面，老年人该看不清还是看不清。而Cool UI 5.7则是完完全全的一种模式，启用这个模式之后彻头彻尾的改变了所有界面的布局。

    酷派大神F2在配置方面采用了一颗八核1.7G的MT6592处理器，我们知道MT6592是不支持4G网络的，所以在基带方面则加入了MT6590的基带芯片进而提供了对4G网络的支持。在运存方面大神F2也比较厚道的配备了2G的运存，同时内置的存储空间为16G。这个配置相对于目前以骁龙801为主的旗舰产品来说并不算高，但是在千元左右的机型内还是有着一定的优势。

     虽然安兔兔跑分在真正意义上代表不了什么，但是还是在某种程度上可以拿过来做一下参考，安兔兔的跑分成绩在32000左右。这个分值也比较符合这类机型的成绩。跑分仅是一个参考，而实际还要看游戏的运行体验。

     由于现在安卓手机现状是硬件过剩，所以大神F2的这套配置在理论上说是可以满足大型游戏的运行的。所以测试直接选用了3D较对硬件要求较高的NBA2K14以及极品飞车17进行测试。应该说测试的结果还是比较满意的，整个游戏运行下来没有明显的卡顿。于是也没有必要对小型游戏进行测试。所以大神F2这套配置满足常见游戏以及软件还是完全没有问题的。

    酷派大神F2采用了一颗1300万像素的主摄像头，具体拍照表现我们直接看样张。

    从样张中不难看出大神F2的色彩还原能力是不错的，而且照片的通透度和锐度也相当的高。F2.0的大光圈能营造出很好的浅景深效果。同时1300万像素的对于物体的细节捕捉还是十分到位的。从最后一张夜景拍摄中不难看出即便是光线复杂，但是整体白平衡控制的也是比较不错，色彩还原真实，只不过在暗部会有一些噪点的出现。整体看来这颗1300万像素摄像头的表现还是可以圈点的，对原片感兴趣的朋友可以关注我的新浪微博@唐山雨晨 私信索取。

    ok，经过一段时间的体验大神F2给我留下印象最深的是出色的是整体的综合表现，虽然没有旗舰的配置但总体配置仍能满足用户的需求；虽没有超薄的机身，但后壳弧度的处理以及四角椭圆倒角手感非常不错，搭载成熟稳定的Cool UI 5.7系统使得F2在系统方面易用度和实用度更具优势。总体来说定位于999元的大神F2我们不能以对待几千元旗舰机的挑剔眼光去看它，实实在在的配置与系统能够满足用户各方面的需求以及对于4G网络的支持才是它的亮点所在，整体虽平淡却有着不平凡的表现，这就是酷派大神F2。
