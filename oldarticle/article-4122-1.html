醉翁之意不在酒 - SONY 防水运动 MP3 NW-WS413

13022 0
如果让我推荐运动耳机, 也许曾经的我会不假思索的回答 森海 MX685 或者 MX686, 实际上热爱
健身的我却很少佩戴耳机进行运动, 除去伴随一些剧烈运动的流汗等问题以外, 稍不小心拉扯到耳机
线材也是常有的事, 然而一再点歪 "科技树" 的 SONY 似乎早已考虑到这些问题.

不知道 WS413 是否该为特殊的 Walkman 系列, 它既耳挂式的入耳式耳塞本身也是一个 MP3,
初接触此系列的产品, 根据网上的信息来看, 其似乎是 W273S 的续作.

[attach]210920[/attach]

WS413 为绕耳佩戴, 产品造型摆放呈现为心形,
L/R 左/右 分别以灰色和红色区别在左边耳挂部分设有 "盲点".

[attach]210917[/attach]

尽管 WS413 体积小巧, 但是功能按键一应俱全 - 右边: 开关按钮, 暂停/播放, 歌曲 前/后 切( 长按
为 音乐 快进/快退 ), 右边: 歌曲循环模式切换, 环境音模式开关 ( 由于此产品内置麦克风, 运过程
中按下无需摘下耳机跟他人沟通 ) 音量增减.

单击 mode 键: 正常播放 / 重复播放 / 随机播，长按 mode 键: 按文件夹播放 / 按专辑播,
双击 mode 键: 单曲循环，环境音键: 三种模式, 不同程度的大小, 环境音量 A / B / C 逐步增大

[attach]210918[/attach]

产品参数

接口: USB 2.0
频响: 20 ~ 20000Hz
电源: 内置可充电锂离子电池
充电时间: 1.5 小时 / 快速充电: 3 分钟可支持播放长达约 60 分钟
工作温度: -5℃ 至 45℃
质量: 约 32g
耐水性/防尘: IP65/IP68 ( UEC 60529 ) 相当, 需要游泳耳塞 ( 内附 )
容量: 4GB / 实际可用: 3.2GB
续航时间: ( MP3 128Kbps ) 环境模式开: 约 8 小时 / 环境模式关: 约 12 小时
( 线性 PCM 1411Kbps ) 环境模式开: 约 7 小时 / 环境模式关: 约 10 小时
配件: 耳塞 / 约束带 / 游泳耳塞 / USB 底座 / USB 连接线 / 启动指南 / 说明书 / 注意事项

[attach]210921[/attach]

支持音频格式

MP3: ( MPEG-1 Layer3 ) 32-320Kbps, 支持 VBR, 采样率: 32 / 44.1 / 48KHz
WMA: ( ASF ) 32-192kbps 支持 VBR, 采样率: 44.1KHz
线性PCM: ( Wave-Riff ) 1411Kbps, 采样率: 44.1KHz
AAC: ( mp4 / m4a / 3gp ) 16-320Kbps,
支持VBR, 8 / 11.025 / 12 / 16 / 22.05 / 24 / 32 / 44.1 / 48KHz
总体来说, 虽然不支持诸多发烧友偏好的 ape, wv, flac, tak, tta 等无损格式,
但对压缩音频格式并不算太窄.

[attach]210919[/attach]

没有屏幕的缘故, 耳机少部分功能切换时以英文女声提示.

[attach]210922[/attach]

抛开众所周知的 ZX1 / ZX2 系列, 同样有着 Walkman 标识的 WS413 算得上是一款特殊的产品.

[attach]210923[/attach]

[attach]210924[/attach]

包含自带, 由小到大共 8对 耳套, 其中白色为常规硅胶套, 黑色为 游泳耳塞.

[attach]210926[/attach]

靓丽的橘黄与绿色, 不免让人联想到 SONY 曾经的笔记本 VAIO 系列.

[attach]210925[/attach]

拿下硅胶套的耳塞滤网部分.

*IPX8 ( 持续浸水防水等级 ): 经过测试, 当浸于 2 米深的水中约 30 分钟时, 安装有游泳耳塞的
Walkman 仍可以保持其操作性.

IP6X ( 灰尘防护等级 ): 经过测试, 在含有最大直径约 75 微米的灰尘颗粒的测试设备中晃动约 8 小时
后，安装有游泳耳塞的 Walkman 仍可以持续防尘.

防水性能仅使用于 淡水, 自然水, 汗水, 泳池水和盐水，防水性能在上述五类水之外的其他液体环境下
均不适用.

[attach]210927[/attach]

充电底座与数据线, 一如既往, SONY 并不配送充电器.

[attach]210928[/attach]

连接充电底座的 WS413 - 充电状态下 黄绿灯交替闪烁

[attach]210929[/attach]

"密不透风" 的游泳耳塞.

[attach]210930[/attach]

在网上个人也得知 WS2xx 系列早已名声在外, 所以毫无顾虑的将其浸泡于水中拍摄.

[attach]210931[/attach]

耳塞部分放入耳道内舒适度一般, 但由于特殊的双耳挂绕耳佩戴结构, 即使对于眼镜党来说 WS413
并不会出现压耳或夹耳的问题.

测试曲目:

[attach]210934[/attach]

Zard - <<負けないで>> 同名单曲专辑,
单击试听酷我音乐.

    其实 坂井泉水 并非是专门演唱动漫曲目的歌手, 不过其演唱的<<灌篮高手>>ED04, 柯南 OP04
广为人知. 而<<負けないで>>曾是<<白鸟丽子>>的主题曲, 出生年代的关系, 对那个时代的连续剧
我无从谈起.

    - 作为一款主打防水与多功能的 MP3 和耳塞而言, WS413 解析谈不上出色, 在女声表现上没有
明显修饰, 中高频谈不上亮眼, 作为 J-Pop 类型的曲目<<負けないで>>给我个人的感受稍带有几分
摇滚风格, 93年发布的音乐专辑, 相较今天的作品而言由于录音质量的关系, 中高频部分相对短板,
然而在坂井泉水口中, 此曲 "铿锵有力", 正如其主题<<不要放弃>>.

[attach]210935[/attach]

MAKE-UP- <<Pegasus Fantasy>> , <<Saint Seiya 原声大碟>>
单击试听酷我音乐.

   "小宇宙, 圣衣, 雅典娜, 天马流星拳, 庐山升龙霸"... 是我们这些90年代成长起来的动漫迷最熟悉不过
的几个名词. <<圣斗士星矢>> 作为首部登陆中国的日本现代漫画, 以华丽的圣衣, 热血的人物, 宏大
的神话背景在刚刚开始接触动漫的中国青少年心目中留下了深刻印象.

     主题曲 <<Pegasus Fantasy>> 整曲慷慨激昂, 不过 WS413 乐器分离稍显短板, 整曲表现稍显
"平面", 个人认为 <<天马座的幻想>> 亦是非常有代表性的日式摇滚 与 电子乐的完美结合, 其实
伴随着这样的音乐进行健身非常合适.

------------------------------------------------------------------------------------------------------

总结

对于更多的关注传统耳塞以及随身播放器的我而言, 既是 MP3 又是入耳式耳塞的 NW-WS413
确实是 Walkman 家族稍特殊的一款产品. - 其入耳式部分, 中长时间佩戴仍有少量不适感, 不过由于
其特殊的心型绕耳佩戴结构, 即使对于 "眼镜党" 而言, 它倒并不会出现压耳或夹耳的情况.

-5℃ 至 45℃ 环境适应, 防水, 防尘 等功能, WS413 可以适应滑雪, 游泳, 甚至在洗澡时佩戴, 以及
3分钟快速充电, 环境音等贴心设计, 功能与适应范围而言, WS413 是一款非常出色产品.

当然, 如果其不再是仅仅只有 4GB / 8GB 两种容量可选, 声音水平还能更上一层楼的话, 对于更多
苛求音质的用户而言, WS413 将会是一款让人 "醉翁之意不在酒" 的杰作.
