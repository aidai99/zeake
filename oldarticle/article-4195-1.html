机皇再临 三星Galaxy Note 7爆料汇总

14921 5

早前，三星曾经公开发表邀请函，称其将于2016年8月2日发布旗下全新的旗舰手机产品Galaxy Note 7。虽然三星竭力在发布会之前将Galaxy Note 7之规格配置进行保密，但是终究是抵不过诸多爆料大神们的“毒手”，而也正式得益于这些爆料大神们，我们得以在三星正式发布这款旗舰手机之前，一窥其强大规格。

一、        不再有edge+，三星让曲面屏也有S-Pen加持

在去年推出试水性质颇为浓厚的Galaxy S6 edge+之后，三星今年明显是对产品线做出了精简。而在今年的Galaxy Note 7上，三星似乎有意将其打造成三星旗下智能手机的集大成者。






根据目前网上流传的真机照片以及渲染图，我们可以相当确定Galaxy Note 7将会配备一块5.7英寸的2K分辨率Super AMOLED双曲面显示屏。同样为了增进手感表现，三星也为Galaxy Note 7配备了和Galaxy S7 edge设计类似的曲面玻璃后盖。机身颜色则提供黑、银以及蓝色三种。

至于Note系列招牌的S-Pen，三星这次自然不会落下。根据相关爆料，Galaxy Note 7的S-Pen将会提供屏幕录制以及GIF动画生成功能，使得这支“笔”更加强大。

二、        依旧双版本，依旧是最强的性能表现


作为高通的最大客户以及竞争对手之一，三星的旗舰智能手机每年都会用上最强的SoC。根据目前曝光的信息，三星Galaxy Note 7将会在SoC上提供高通骁龙821以及Exynos 8893两种版本，分别对应不同地区。包括中国大陆、北美在内的地区将会提供骁龙821版本，而Exynos 8893版本则会在欧洲、亚太等市场发售。


不过较为奇怪的是，三星并未给Galaxy Note 7提供早前传闻之中的6GB RAM。根据最新的准确消息，三星Galaxy Note 7依旧“仅”提供4GB RAM，如此自然是让不少期待6GB RAM的发烧友们失望不已。不过三星会在Galaxy Note 7上搭载全新的UX用户界面，其全新的用户界面在经过精简之后，应当会有利于进一步挖掘硬件性能表现。

最后，三星Galaxy Note 7的UFS 2.0机身内存容量将会直接由64GB起跳，并提供128GB的版本。而两个版本都将会支持SD卡拓展内存，相比目前市面上主流的32GB 内置存储手机来说倒是显得相当厚道。

三、        三星首款Type-C设备，但却只有2.0标准


在USB Type-C接口慢慢普及的今天，三星对于这一全新的接口似乎没什么热情。不过在Galaxy Note 7上，三星有望使用这一全新的接口。不过根据早前Galaxy Note 7数据线的曝光信息，Galaxy Note 7应当还是只支持USB 2.0标准，而不是最新的USB 3.1标准。

四、        噱头十足的虹膜解锁，实用性有待考证




除了目前旗舰智能手机标配的指纹识别解锁，三星Galaxy Note 7本次被曝还提供了虹膜解锁功能。通过在机身正面上方搭载专用的传感器，三星Galaxy Note 7能够对用户眼中的虹膜进行扫描并进行包括解锁手机在内的安全认证操作。

和早前其他手机上的，使用前置摄像头完成的眼纹识别不同，专用的虹膜解锁传感器对于光线的要求并不高。在早前我们评测Lumia 950XL时，我们便发现Lumia 950XL可以在相当昏暗的环境下成功识别虹膜并解锁。而三星Galaxy Note 7的虹膜识别功能在体验上应当会是大同小异。

五、        足够强的相机与电池，但是并没有“更强”


和去年的Note 5一样，三星本次选择为Galaxy Note 7配备了和上半年旗舰手机完全相同的相机硬件。Galaxy S7、S7 edge上的后置1200万像素双像素光学防抖相机在上市之后获得了用户的一致好评，所以我们自然也有理由相信Galaxy Note 7的拍照素质至少能够和Galaxy S7系列持平，当然不排除三星能够在软件方面做出进一步努力，使得Galaxy Note 7的拍照素质能够略胜于Galaxy S7系列。

如果说相机规格完全相同尚算是能够让人接受的话，那么Galaxy Note 7配备的3500毫安时电池便会让许多人感到不解和不满了。虽然其支持快速充电以及无线快速充电依旧很有“旗舰范”，但是相比S7 edge更小的电池容量则不免地让人担心其续航能力。

以上便是目前所有的，关于三星Galaxy Note 7的爆料信息，以上信息也基本涵盖了Galaxy Note 7的所有特性。从规格配置上来说，三星Galaxy Note 7无疑是今年下半年亮相的，硬件配置最为强悍的智能手机之一。

而作为一款真正的年度超级旗舰，三星Galaxy Note 7的售价自然不会便宜。根据外媒曝光的消息，三星Galaxy Note 7在欧洲的售价将会高达849欧元，约合人民币6312元。在国内，行货版Galaxy Note 7的售价也可能会高达6288元。

三星Galaxy Note 7将会于今年8月2日正式发布，届时我们也将会继续报以关注。而关于这款旗舰手机的详细规格配置也将会在那是全部揭晓，不知道三星会否给我们进一步的惊喜呢？
