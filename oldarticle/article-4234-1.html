三星Galaxy Note 7上手：一台优秀的年货

18062 6

离离原上草，一年一台Note。三星今天正式发布了其全新的Galaxy Note 7智能手机，这款全新的智能手机将会在8月19日在美国正式上市。而在这款新机正式上市之前，我们也获得了提前摸到它的机会。在简短的上手之中，有一点逐渐变得明显：三星也许并没有将Note系列的套路颠覆，但Note 7依旧代表着逐年进化而来的终极品质。

美爹版的Galaxy Note 7内置的是四核心的高通骁龙820处理器（国行版估计也是），搭配Adreno 530 GPU以及4GB LPDDR4 RAM。总的来说Note 7搭载的性能硬件和Galaxy S7、S7 edge完全一样。在45分钟的上手之中，这款手机运行的速度快得堪比香港记者，而随着使用Vulkan图形API的游戏们逐渐上架，Note 7的强悍会在日后更显明显。

Note 7和S7系列的相似度并没有就此打住。Galaxy Note 7配备了和S7系列相同的1200万像素双像素相机，于是Galaxy Note 7便跻身于市面上最顶级的拍照智能手机之间和它们谈笑风生——尤其是在低光照环境下。三星同样还让Galaxy Note 7拥有了IP68级别防水防尘，而在我们看来几乎这应当是一个今后旗舰智能手机都不可或缺的特性。Galaxy Note 7那块惊艳的5.7英寸双曲面Super AMOLED显示屏在左右边均有弧度，在这一上面三星也有提供edge边缘交互界面，以便用户快速开启APP或找寻联系人。哦，Note 7另外还支持了256GB的SD卡拓展内存——若是你觉得起步的64GB内存依旧不够用的话。


总之，Galaxy Note 7和Galaxy S7系列在规格上的重合并不算那么的令人惊奇，这也无碍Note 7成为目前市面上最顶级配置的智能手机之一。而在这里，笔者想要先钦定Galaxy Note 7为目前握持感最舒适的Galaxy Note手机——甚至比S7系列两兄弟们握起来都更加舒适。今年三星追求硬件设计上的对称，于是Note 7的前后玻璃均提供了弧线，这些弧线自然也让这台手机能够更加舒适地被握持于手中。而也许正是因为如此的设计语言，反倒使得工程师只能够为其配备3500毫安时的电池，比S7 edge还要小些。

关于Note 7的续航能力，我们会在后续的详细评测之中进行探究。但至少目前值得庆幸的是，三星尽管走上了Type-C的道路，但并没有像摩托一样选择砍掉Note 7上的3.5mm耳机接口。

那么Note 7究竟有啥不同呢？


首先，Note 7提供了骚气得不行的蓝色机身+铜色金属中框的机身配色。这个蓝色有点像蒂芙尼蓝，并且看上去相当迷人。

除了外表之外，Note 7还搭载了虹膜扫描仪。只要给它一个眼神，扫一眼屏幕，Note 7即可解锁或是让你访问你早前加密的手机内容。这颗传感器首次使用需要大约1分钟来录入虹膜信息，而在录入成功之后，即便是戴着长者一般的眼镜也能够毫无困难的解锁（三星表示某些眼镜上的特定涂层会使得虹膜扫描仪识别率降低）。正如所有科学技术一般，在一开始这个小东西并不显得那么有用，在我简短的上手之中，Note 7识别我眼睛所用的时间还是要比直接用指纹识别解锁要长那么一些。也许当下这个虹膜解锁用途有限，但是三星表示在将来我们将会可以使用虹膜登陆账户，甚至对Samsung Pay进行支付授权。


至于Note 7的内容加密功能，则很明显是为了商务人士们所进行的考量。三星本次加入了一个所谓的“安全文件夹”，在此之中用户可以储存文件甚至应用程序。三星同时还进一步精简了TouchWiz用户界面，使得其更拥有谷歌的Material Design风格。而进一步深挖时，我们还会发现相机界面也获得了精简，在移去一列拍照模式按钮之后，三星选择了使用滑动进行模式的切换。Note 7的相机本次还支持HDR视频流。


另外，三星还将所有S-Pen相关的APP集合到了一个叫做“Samsung Notes”的应用之中，用户在此可以储存手写的笔记和绘画，编辑图片什么的自然也是不在话下咯。

说起S-Pen，Note 7的S-Pen的外形和去年Note 5上面的似乎并没有什么不同，但是Note 7的手写笔是确确实实和手机本体一般拥有IP68级别防水防尘的。结合上防水的Note 7本体，你甚至可以在水下拿出Note 7的手写笔挥洒一番创意——尽管我怀疑会不会有人真的这么玩。使用这支笔在Note 7的屏幕上书写也变得更加自然，而这一部分是因为三星本次将笔尖大小缩小至和真正的标准原子笔尖一样的0.7mm。

S-Pen本次也拥有了一些全新的特性。你可以使用S-Pen将屏幕上显示内容放大至多3倍，或是进行实时翻译（如果你要是想浏览英文版的知客数码，这一特性应该是挺有用的），尽管这翻译功能的翻译依旧停留在机翻水平。这一功能需要谷歌翻译进行支持，于是其智能程度自然也和谷歌差不多（到国内可能会换成百度吧……细思恐极+画美不看）。

相当不幸的是，S-Pen的悬浮翻译功能目前仅限于单词翻译——我早前还想着能够一整句或是一整段地丢给手机来翻译呢，而三星再后来则粉碎了我的YY。


S-Pen本次最好玩（至少是最重要）的新特性则是在屏幕上截取一小部分区域，生成GIF动画了。虽然这一功能的工作原理简单得令人发指，但是在生成GIF之后，你甚至可以选择这一GIF循环的方式或是在上面做些极好。Note 5上广受好评的息屏速记功能在Note 7上也依旧提供，同时结合Note 7的常亮屏幕，你也可以将笔记固定在AOD之上以便查看。

最后，那些手持Note 5的用户们值不值得买一台Note 7呢？抱歉在这儿我说不清。创新来源于适应和创造，而我总是在想三星还能够对Note系列做出什么改进，以使得它们无可替代。那些期望在Note 7上找到巨大变化和创新的人们也许会失望，但以目前的状态来看，Note 7整体依旧是相当一颗赛艇的。
