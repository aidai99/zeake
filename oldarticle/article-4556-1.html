索尼Xperia X Compact上手：一台很mini的Compact

18236 5
IFA大会上索尼发布了两款智能手机，其一为其下半年旗舰Xperia XZ，而其二则是传闻之中的小屏机器Xperia X Compact。这款小屏机的外观类似于通常发布的Xperia XZ，但是索尼将之称为Xperia X Compact，所以其并没有搭载和XPeria XZ一样的高规格硬件。




Xperia X Compact最先让人注意到的自然是其外观设计，如今Xperia X Compact终于算是一定幅度地脱离了所谓之全平衡设计——尽管从某种程度上来说这货脱离的程度并不彻底。其采用的全新LoopSurface设计风格包含了对称的圆弧边框以及平整的上下部分，给人感觉非常地像是一位工程师设计出来的产品，单实际上相比那些一体化机身设计，给人的感觉还是相当耳目一新的。

Xperia X Compact的整个机身材质都是塑料，和Xperia XZ的金属后背截然相反。但是Xperia X Compact的塑料机身更加轻巧，抗摔性能和耐久度也更好。所以倒也不过过于羡慕XZ的金属机身咯。




这款机器抓在手里的感觉还是挺不错的，这部分来源于其较为小巧的机身体积，但其聚碳酸酯机身材质以及两侧的机身弧线自然也是功不可没。X Compact给人感觉很扎实，但又很轻，所以单手操作自然是不在话下。所提供的淡蓝色、黑色和白色款式搭载不同的后背处理手法，亮光版本的后盖在吸指纹上自然会比磨砂后盖明显得多。

Xperia X Compact搭载了一块4.6英寸的HD分辨率特丽魅彩LCD屏幕，正面覆盖有2.5D玻璃。内置高通骁龙650六核心处理器、3GB RAM以及32GB ROM，支持内存卡拓展。虽然配置令人失望，但如此其内置的2700毫安时电池自然也是可以提供不错的续航能力。除了SoC之外，其后置2300万像素相机以及3个附加传感器的设定等等配置则和Xperia XZ完全相同。但是需要注意的是其正面摄像头则缩水至500万像素。




另一个让人遗憾的是Xperia X Compact的机身并不具备防水能力，所以若是今后你闲的蛋疼买了一台这货，也请千万记得不要将它像你以前对待索尼Xperia手机一般投入水中。但至少从好的一面来说，Xperia X Compact支持高通QC3.0快充，并搭载USB Type-C接口。

Xperia X Compact的机身三围和重量和Xperia Z3 Compact相近，但其机身厚度略有增加。至于NFC功能以及3.5mm耳机接口这种日常需要的功能，索尼自然也不会欠奉。在发布会上索尼还拿出了几个官方原厂的翻盖保护套，还是挺值得一看的。




机身后背上那颗2300万像素的相机自然会是Xperia X Compact最重要的卖点。尽管这款机器的前置相机并不如Xperia XR那么诱人，但是其后置的白平衡传感器、激光辅助自动对焦以及和XZ看齐的5轴电子防抖应该还是能够让粉丝们略感满意。当然啦由于SoC性能的大幅缩水，Xperia X Compact自然也是不支持4K视频录制的了。

有些人在看习惯了之前的索尼Compact系列产品之后，对于X Compact在规格上的疯狂缩水自然是深感不爽。但至少Xperia X Compact并不叫做Xperia XZ Compact，所以其搭载和Xperia X几乎相同的硬件自然也是情理之中。索尼顶尖的拍照功能，最新的外观设计，加上中端的配置，整体来说并不算糟糕啦。


Xperia X Compact遵循着诸多之前Compact系列智能手机留下来的路线，但是同时又向着其他的方向进行了一点探索。在目前售价未知的情况下，我们难以断言Xperia X Compact在面对300美金左右的智能手机产品时能有多少优势。而本身在规格上 并不占太多上风的Xperia X Compact，正好又摊上了一个喜欢漫天要价的爹，所以前景应该并不算光明。

Xperia X Compact预计会在9月25日正式上市，上市时间视具体市场而定，至于价格则有望在接下来的一两周之后公布。
