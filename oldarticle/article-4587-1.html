moto Z&Moto Mods简评：如何重新定义模块化手机

20304 2

早在今年六月举行的Tech World上，联想就正式发布moto Z。按照惯常来说，在智能手机新品如过江之鲫的高频率亮相下，产品热度在经过时间的消磨后要消退不少。但颇为令人意外的是，即便是到现在，moto Z依然是不少科技爱好者所关注的重点项目，并且兴致盎然地等待国行版本的正式发布。

这并不是因为其外形设计有多少优秀，也不是因为硬件配置有多么超前，更不是价格有多么高的性价比，而是其不同于主流设计的模块化设想。




虽说早已经在TechWorld的现场把玩过这款机器，但即便如此，在经过三个月的时间后重新上手时，依然由衷地感叹其做工之精致，质感之优秀。或许应该这么说，moto Z的外形设计并不见得能偶获得多少人的喜欢，亦或者是接受。若不是国行版本仅推出的黑金色配色，将正面诸多传感器“隐藏”起来，其正面的麻子脸比起moto X极不见得好到哪里去。

而是下巴区域的设计，更是让人无力吐槽的是。诸多传感器的存在使得其下巴的宽度相当明显，连那本应是喜闻乐见的指纹识别功能，也因为传感器的设计与功能，显得相当突兀。再算上“moto”的标志，整体下巴数量堪比HTC鼎盛时期。

但即便是再不喜欢这外形设计，当你亲手拿起moto Z时，心底里冒出来的评价，都会是“疑，这质感不错”、“做工精致”等正面的评价。



无论是奇葩的造型设计，亦或者是精妙地令人拍手称好的机身结构，还是说多个传感器所实现的Moto Action功能，以及旗舰级的硬件配置和大幅提升的拍摄能力，这些在某些机型上都能够成为亮点的特性，在moto Z上都显得不值一提。原因也很简单，就在于其背面所有的16个金属触点，也就是模块化设计——Moto Mods。

目前其正式发布的Moto Mods（魔磁）有JBL SoundBoost、Moto Insta-Share Projector、Incipio offGRID Power Pack、Moto Style shell以及Hasselblad True Zoom，分别对应音箱、投影仪、外置电池与无线充电、材质后盖以及拍摄模块。




在Moto Mods中技术含量最低，但必要性最强的，自然是Moto Style shell。其实可以将Moto Style shell看作是Moto Maker的简化版，除国行版本的moto Z将标配两个之外，moto还将会提供多种不同的材质以供选择。这不仅是让每个人的moto Z都显得不一样，从审美以及手感表现的角度来说，当两者结合在一起时，才是这款智能手机的完全体。






至于JBL SoundBoost、Moto Insta-Share Projector、Incipio offGRID Power Pack，也就是扬声投影仪以及电池模块，相对而言并没有太多值得细说的地方。自然是因为Moto Mods的使用相当的简单，将模块与moto Z简单的一扣，就直接进入相关的工作状态，并不需要繁琐的配对或设置。

并且各个模块所能够做到的也是显而易见的事情，并没有多少值得深究的地方。所以关于Moto Mods的部分，要着重介绍的就是首次亮相的Hasselblad True Zoom。





正如Hasselblad True Zoom这个名字所代表的含义，这款Moto Mods是由moto与哈苏合作推出的拍摄模块，其在设计与体验的层面上，能够称得上是Moto Mods的最佳代言人。

当其与moto Z结合使用时，采用橡胶材质所打造的握把设计，能够给予相当不错的握持体验以及稳固性。而那明显的氙气闪光灯与张扬的摄像头设计，都可谓是有模有样。拿出门给好友展示说是哈苏的最新卡片机，估摸能够获得不少的感叹。

并且在Moto Mods的技术下，Hasselblad True Zoom与moto Z的结合使用可谓是天衣无缝。当两者结合时，Hasselblad True Zoom就成为moto Z的后置摄像头，并提供多种场景模式、功能特性以及格式保存。并且在拍摄后的图像传输以及分享等系列动作，都可谓是一气呵成，可不是外置摄像头们能够比拟的。




但得泼盆冷水的是，原以为在外置的形态下，Hasselblad True Zoom能够采用1英寸的传感器，以获得碾压智能手机的成像画质，将moto Z变成一台相机。但实际情况是，Hasselblad True Zoom所采用的传感器也就只能够说是智能手机中的“大底”。

虽说1/2.3英寸的传感器尺寸、1200万像素以及1.55um单个像素面积大小，确实比起moto Z的后置摄像头要好上不少。但也只能够说是智能手机中的中上水准，比起翘楚们可是有着差距。只能够安慰地说，这个模块的设计理念里“实用性”比“画质”的重要性更高。



应该如何评价moto Z与Moto Mods？这显然是个两极分化的问题。崇尚者无疑会对moto Z那精妙的设计与精致的质感表现所打动，也会被Moto Mods那创新的模块化设计所打动。鄙弃者则是会对moto Z那不同于主流的设计语言呛之以鼻，也会对那麻子脸的传感器开孔大加嘲笑。更是会对Moto Mods下以“看似创新，实则食之无味”的评价。

可毋庸置疑的是，moto Z与moto Mods的设计理念都将会成为手机历史中的里程碑，所做出的创新与尝试无疑是极具意义的。但这就能够获得认可、市场的接受，乃至引领潮流么？自然是留待给最真实的消费者所做出的反馈。
