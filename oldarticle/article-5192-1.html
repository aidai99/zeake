从入门到退烧 市售机械键盘大盘点

20061 3
相信大家对第一次使用电脑干什么有着深刻的记忆，但要问及第一次使用的机械键盘的感想时，基本上毫无印象。早期的机械键盘受困于成本和故障率，早已被厂商所主推的薄膜键盘所取代。等我们对键盘有所想法时，早就不知机械键盘到底是什么味道。

但随着近些年电竞业的火热，加上用户们越来越不满意薄膜键盘的手感，本应扫入历史垃圾堆的机械键盘反倒是浴火重生。从之前的阳春白雪，到现在大大方方敞开怀抱，拥抱主流消费者。

机械键盘本身也在不停地进化着，不仅仅拥有比薄膜键盘更好的手感，还有区分手感显著的四种常见轴体——黑轴，红轴，茶轴，青轴，各有各自的拥趸。但难以回避的劣势——售价上，说不上亲民。不过好在我们现在拥有繁多的特惠活动，这里小编就给大家推荐些值得购买不同价位的机械键盘，以供参考。

洛斐 DOT 圆点蓝牙机械键盘 少女心爆棚的复古无线键盘


如果面临着要向女生推荐一款颜值不错的入门机械键盘的时候，相信在很多人心目中的各种入门机械键盘“神器”均惨遭折戟。不过放心，为了拯救机械键盘普遍男性化严重的问题，洛斐站了出来。近日在淘宝众筹的洛斐 DOT 圆点蓝牙机械键盘，给机械键盘普遍大老粗的画风加入一丝柔美曲线。


洛斐 DOT 圆点蓝牙机械键盘外观上最下功夫的，应该就是它特有的打字机设计。不同于市面上机械键盘普遍走复古风老式打字机路线，洛斐对老式打字机造型进行了全新的设计——用抛光处理的塑料外壳给予视觉效果上一种果冻的感觉；键帽采用老式打字机圆形造型，但采用塑料材质搭配上清爽配色，带来一股小清新既视感；键盘本体也进行了重新设计，无边框的悬浮设计看上去就是干净舒服。


洛斐 DOT 圆点蓝牙机械键盘最关键的输入体验上，也是下了一翻工夫。支持蓝牙无线USB有线两种连接方式。轴体采用 GATERON 家的青轴轴体，这是最具有机械感的轴体类型，支持三档白色背光调节。同时为了保障处于蓝牙无线连接下开启背光的续航问题，洛斐 DOT 圆点蓝牙机械键盘直接内置 4000mAh 锂电池，正常开启蓝牙连接背光使用下能达到2年续航。支持Windows、Android 和Apple 家的 Mac 或 iOS 设备，仅需拨动右侧的设备切换键，即可迅速地在不同类设备之间切换。


不过洛斐 DOT 圆点蓝牙机械键盘也有难以回避的硬伤，首先配色上由于太过于小清新化，导致男生要是用起来稍微有点怪怪的，还是适合女生使用。第二就是为了适配 Mac 平台对键盘的键位做出了一定的修改，要是习惯在 WIN 平台下使用的话多少还是有点适应问题。不过考虑其淘宝众筹上仅需329就能拿下粉白款，作为一款入门级机械键盘足够满足女生的需求。

ikbc c87 / 104 入门级搭载Cherry原厂轴键盘


虽说市面上便宜的机械键盘并不少，但大多采用国产轴体，带来的反馈自然不如 Cherry 原厂轴精准的反馈来的舒爽。想要一步到位直接Cherry 机械键盘走起，发现售价难以承受。这不国内第三方机械键盘厂商就瞄准这目标群体，推出售价相对原厂键盘便宜，又同样采用Cherry 原厂轴的机械键盘以供选择，ikbc c87 / 104 就是其中一款。


ikbc c87 / 104 拥有两种版本，87 键和 104 键两种，区别仅仅是有没有数字区而已，桌面空间吃紧的朋友可以选择 87 键位版本。配色上，拥有黑色和白色可供选择。键帽上选择了这个价位少见的 PBT 材质，字体加工却采用镭雕工艺，长时间使用字体颜色稍淡。


轴体上直接使用 Cherry 原厂轴，提供黑轴，红轴，茶轴和青轴以供选择。大键采用卫星轴设计，比平衡杆设计的手感更好。另外搭配 FN快捷键，可以调节音量和屏蔽 WIN 键。背面走线可选择直线和侧边走线，脚撑采用了二阶脚撑设计，两个脚撑上都有防滑脚贴，十分贴心。

ikbc c87 / 104 细致的双阶脚垫设计，加上 Cherry 原厂轴的搭载，提供良好的手感反馈。遗憾的是并没有提供背光以供选择，键盘整体设计普普通通，适合在乎售价的实用党。

CoolerMaster 烈焰枪系列 极光版 将白光玩成极光的键盘


不知道是谁说的：不能玩灯光的机械键盘就不是一把好键盘。看上去还是蛮有道理的，但想要买把灯光酷炫，原厂 Cherry 轴手感加持，售价可都不便宜哇。这不，老牌外设厂商 CoolerMaster 就给我们炫了一把灯光特效键盘，展示一把什么叫做灯光闪花眼。


CoolerMaster 极光版仅有87键位可供选择，喜欢独立数字区的朋友们就要失望啦。采用分离式 Micro USB 接口设计，外壳涂有一层类肤材质，手感不错。轴体采用原厂 Cherry 轴，键帽采用 ABS 涂层透光键帽，整个键帽表面涂层略带磨砂感，手感非常的棒。背面支撑脚垫脚包裹了橡胶垫，打开或关闭撑脚流畅度很不错，有一定的回弹力，仅有一档调节。


CoolerMaster 极光版区别一般键盘最明显就是设置的 FN 键快捷键。F1-F4 配合 FN 键是灯光控制键，分别是灯效开关键、灯效强弱键和灯效模式切换键，F5-F8 键分别是8倍速的延迟速率切换键，F9-F12 为自定义按键，组合 FN 键开启后按下相应的按键就可以很方便的录入自己喜欢的自定义灯效按键，最高可录四种自定义灯效按键，FN 键+PRT SC 键为 WIN 锁定键，在游戏中可避免误操作导致弹到桌面。

CoolerMaster 极光版炫酷的灯光特效是通过官网下载驱动来实现的，目前有陨石灯效，呼吸灯灯效，波浪灯效等等，官网有灯效更新。背光仅有白光，支持五档背光强度调节，可谓是将白灯显示极限到底。


不过 CoolerMaster 极光版也有难以回避的缺陷，灯光特效是玩的很溜，但仅有白光看久了生厌；键盘回馈手感很原厂，但外壳材质采用类肤质涂层非常容易沾灰，清洁不宜。键帽带有点细磨砂处理初上手很棒，一旦使用时长稍久细磨砂被磨损掉手感就生涩不少。如果想要一把灯光足够酷炫的原厂Cherry轴机械键盘，CoolerMaster 极光版足够满足你对灯光的所有想象。

FILCO 87 / 104键 圣手二代 专注按键反馈的低调选手


机械键盘的市场里，总是掺杂着太多外观功能浮夸的产品，真正能专注于反馈打字乐趣的键盘数量稀少。不过好在高端机械键盘里，专注于键盘敲击的反馈和低调朴实的外观还有几款选择，FILCO 87 / 104 圣手二代就是这样一款产品，不算惊艳的外观，靠谱稳定的按键反馈。


FILCO 87 / 104 圣手二代外观上采用窄边框设计，重量不到 1 kg。FILCO 87 / 104 圣手二代采用了全按键无冲突输出设计，但在 USB 接口下FILCO 87 / 104 圣手二代最多实现6按键无冲突，而在PS/2转接口下则可以达到全键无冲。


键帽采用 ABS 材质，同时在外表上镀上一层类肤质材质。轴体采用 Cherry 原厂轴，提供黑轴，红轴，茶轴和青轴以供选择。FILCO 的键盘有很多种不同配色的键盘可供选择，当然也有限量版，这点看个人审美，小编是喜欢奶酪绿多一点。


或许很多人都瞧不上FILCO 87 / 104 圣手二代一眼，它没有酷炫的灯光，也有没方便的宏快捷键，甚至外观设计上都不能吸引下眼球。但FILCO 87 / 104 圣手二代被戏称为送程序猿必备的礼物，已经是对机械键盘手感的最好的标注。如果喜欢朴实无华的外观，日常打字偏多的用户，那么FILCO 87 / 104 圣手二代作为你机械键盘的最终目标是足够了。

Cherry MX Board 6.0 / 9.0 一步到位的旗舰机械键盘


虽然大多数中高端机械键盘基本都是采用 Cherry 原厂轴体，但不同的厂商对 Cherry 轴体的调教还有有所差别。为了能体会到原汁原味的Cherry 轴体的魅力，当然 Cherry 本家出品的键盘更具有吸引力，而其中 Cherry 高端的代表 MX Board 6.0 / 9.0 成为高端机械键盘中的代表。

Cherry MX Board 6.0 是一把令人惊喜的机械键盘，消费者长久以来对 Cherry 键盘的“顽固”感到无力吐槽。咋想到 Cherry 突然一口气就把高端机械键盘该有的全配齐了——全背光、金属外壳、加上第一款采用RK（具有1毫秒响应和真正全键盘无冲突）技术的机械键盘，Cherry MX Board 6.0 作为高端机械键盘诚意足足。


Cherry MX Board 6.0 标配有手托，主要材质是塑料，正面材质为防滑橡胶材质，同时为稳固和键盘的连接，手托与键盘本体采用磁力吸附设计。配色上仅有银黑可选，键盘主体是银色，键帽是黑色。键盘采用金属材质设计，而键帽采用 ABS 材质，并进行 UV 覆膜，表面有类肤质触感。轴体采用全新的 CHERRY MX 轴体，键帽背光仅有红色和部分蓝色切换，这对被各大厂家“跑马灯”养刁胃口的用户们似乎有点不太习惯。


当然，Cherry 也给喜欢更加酷炫的灯光效果的用户提供 Cherry MX Board 9.0 。Cherry MX Board 9.0 的画风和Cherry 严重不符，咋看都不像是一款向来刻板的德国佬出品的设计。足够满足用户们长久以来对高档机械键盘的所有想象——RGB背光、媒体键、宏设定、多灯光模式和跑马灯等。

似乎这么高端的键盘并不适合普通的打字输入？Cherry MX Board 6.0 确实不是针对普通用户而设计的键盘，它的目标用户应该是去追求电竞极致的选手们， 无论是RK技术和全新的CHERRY MX轴体显著倾向于专业电竞选手们的需求。

而Cherry MX Board 9.0 更像是顺应市场的自我革新之作，适合喜欢酷炫外观和多种灯光特效的朋友。如果你想要极致的电竞外设体验，不在乎灯光的绚丽和外观上的低调，那么Cherry MX Board 6.0 作为你的“武器”是足够有资格担当。想要外观和灯光时，Cherry MX Board 9.0 作为终极神器绝不跌份。

总结

从入门到高端，机械键盘不仅手感上有着巨大的差距，外观、灯光特效各有特色。喜欢炫酷灯光的，海盗船雷蛇之类花样繁多的灯光特效足够满足你。喜欢精准按键反馈手感的，Cherry 原厂键盘和 FILCO 87 / 104 圣手二代之类的就够了。键盘从仅仅负担起字符输出的功能后，再次赋予了新的意义——提升使用键盘的幸福感，不正是机械键盘的全新意义么？
