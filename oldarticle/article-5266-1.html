努比亚miniS评测：是Super，更是Surprise

26757 5

 

在半年的时间后，努比亚并不意外的端出“S”后缀的升级款产品——努比亚miniS。更强的性能、更好的拍摄能力以及更长的续航时间，这些都不意外地其身上出现。可比起自家旗舰还要优秀的金属一体化机身、新奇特的机身配色以及极为给力的性价比，可就是相当惊喜。


“S”并不仅仅是过渡

 

 

不同于别家的“S”机型大都是过渡时期的小修小补，努比亚miniS更像是nubia Z13系列的提前阅兵，其质感与做工表现甚至是要超越努比亚现在的当家旗舰。不同于nubia Z11 mini的金属中框与塑料材质后盖，努比亚miniS所采用的金属一体化机身设计，极为明显地提升做工档次与质感表现。

 

 

 

在这样的基础上，努比亚在机身配色上做的尝试就更为喜闻乐见。除代表着高端旗舰的黑金色外，努比亚miniS还带来新的卡其色，既是新鲜感，也是给这款仅1500元的机型带来心理感受的提升。除此之外，努比亚那标志性的红圈Home键与红圈摄像头自然是不会缺席。相比之下，那经过特别设计的纳米注塑天线带更像是当下的流行元素，反而不值得提及。

 

 

在细节方面，采用金属一体化机身设计的努比亚miniS，不可免俗地用上高光切边来表现金属质感。或许是凑巧也或许是努比亚的故而为之，高光切边、塑料缓冲夹层与2.5D玻璃形成相当流畅的过渡，手指从机身边框滑先屏幕的触感相当流畅。

 

也同样是凑巧或故而为之，努比亚miniS并没有在背面用上高光切边的处理。在其内置3000mAh电池的大度量下，背面呈现的饱满曲线，给予握持时的手感相当的舒适顺畅。结合正反面来说，我们给予努比亚miniS的手感表现评价，甚至是要超越无边框的nubia Z11。

 

mini的定位，合适的尺寸

 

 

曾几何时，标榜着mini的机型的屏幕尺寸悄然地突破5英寸。努比亚miniS采用一块1920*1080的5.2英寸屏幕，标称能够提供450nit亮度、85%的色彩饱和度以及1500：1的对比度。此外屏幕还采用in-cell工艺以及2.5D玻璃，并采用康宁第三代大猩猩玻璃。就账面参数来说，努比亚miniS这块屏幕并没有什么特别之处。

 

显示功能方面，也就是标准的色彩与白平衡调整。当然啊，我们更为“贪心妄想”地希望努比亚miniS能够用上努比亚那标志性的无边框设计，这才能够让努比亚miniS的机身尺寸足够“mini”嘛。

 

 

 

 

 

在实际观感方面，努比亚miniS并没有惊喜之处，都是按照常规的水平表现。足够的色彩表现力以及优秀的可视角度，虽算不上优秀顶尖，但说得上是中人之姿。其450nit的屏幕亮度，足够在户外时使用。但努比亚miniS的屏幕最低亮度在夜间使用时，依然是说得上耀眼的，也只能寄望于系统更新功能能够改善。

 

nubia UI：更好看、更好用、更简洁

 

 

努比亚miniS自然是搭载自家的nubia UI，版本号也终于来到最新的“4”。相比起nubia Z11 mini所运行的nubia 3.9.X而言，nubia UI 4在界面设计与功能上并没有多少的变化， 主要在细节以及特性上做到更好的表现与增强实用性。

 

 

 

 

对于nubia Z11系列的机型用户来说， nubia UI 4已经是相当的熟悉，即便是努比亚品牌的新用户，也能够在这讲解明了、逻辑清晰地轻松上手。但对于nubia UI的老用户来说（比如我这用nubia Z9跳上来），nubia UI 4的变化与进步可就是泪流满面地终于等到你来。

 

在nubia UI 4里，应用商店、主题商店以及音乐都进行重新的设计以及资源整合，特别是主题商店不再有网页跳转的感觉。此外其还加入喜闻乐见的红包助手、nubia闪传、nubia漫游以及云备份，自然是给使用体验带来不少的进步。

 

 

可比起这些常规的设定，nubia UI 4里的特性功能更为值得大书特书。分屏功能并不是nubia UI 4的独有特性，但不同于别家厂商需要在特性的场景、特性的应用以及特性的条件下才能够实现，只要你需要，努比亚智能手机就能够实现分屏。配合应用分身功能，自己给自己发微信这样的孤独事情也不会是问题。（可谁会这么孤独呢？）

 

 

截图功能进一步升级为“超级截图”，能够自定义截图形状、进行长截图或录屏操作。值得赞赏的并不是其功能繁多，而是触发极致相当的巧妙。nubia UI 4能够根据截屏组合键的按压时间，分别进行单一截屏或超级截图。此外还能够通过长按指纹识别以开启超级截图。简单的地说，截图既是简单轻松，也是更多选择。

 

 

虽说努比亚miniS并没有nubia Z11的无边框，但那有趣又实用的边框手势可没有落下。通过不同的动作与场景，努比亚miniS能够支持亮度调整、单手模式、快速返回等功能。其中最为实用的，就是通过边缘滑动切换应用，像是翻书那样简单高效。

 

意外的提升：进步的拍摄能力

 


换用金属一体化机身设计与新的机身配色，自然是给努比亚miniS带来质感的飞跃进步以及体验感受的变化。但可别忘记拍照这事情，这可是努比亚一直标榜的功能特性。特别是在自拍这事情上， 努比亚miniS带来的惊喜可是要超出意料的多。

这可不是在说nubia Z11 mini的自拍不够意思，而是努比亚miniS的进步相当难得。其所采用的1300万像素索尼IMX258传感器，即便是用在后置摄像头上也足以担当大任。新的传感器不仅带来更高的像素，更是使得努比亚miniS在自拍时能够支持PDAF混合相位对焦，让自拍要更为方便的多。

但需要注意的是，努比亚miniS取消此前的前置补光灯设计，改为以屏幕瞬间高亮度的形式，至于好坏与否就是见仁见智的问题咯。



当然自拍可不是靠硬件堆砌出来的，还得软件的辅助才能俘获少女们的欢心。努比亚miniS同样支持10级美颜调节以及9种美颜滤镜，在软硬结合下，自拍的效果自然是妥妥的。


从“能拍星星的手机”再到“手机中的单反机”，努比亚一直将拍照标榜为自家的特性，并持续地在软硬件上投入研发，努比亚miniS自然不会让人失望。其后置摄像头采用2300万像素的索尼IMX318传感器，并配备F/2.0光圈并支持PDAF混合相位对焦技术，在同价位机型中，是足够豪华的配置。

若是对努比亚的机型足够了解，能够发现其会在同系列机型中采用相同的传感器，但通过光学防抖、功能特性以及细节来区分产品定位。所以努比亚miniS率先采用的索尼IMX318传感器，更像是一次提前练兵，也体现出这款“mini”机型一点都不mini。

当然也得说明，相比起此前的IMX298传感器而言，努比亚miniS的IMX318传感器的单个像素面积从1.14um下降到1um。在画面的硬素质上是略显退步，但差距并不会太大，换来的更高像素有着更好的解析力。


相比起干巴巴的硬件参数，我更为愿意说下努比亚miniS的拍摄功能，因为实在是好玩又简单。其所搭载的NeoVision 6引擎，提供多种拍摄与特性支持。

除常规的全景、微距、DNG模式外，还有努比亚特有的光绘、星轨、电子光圈，更有好玩有趣还简单的克隆相机、延时摄影以及手持长曝光，通过简单的操作即可实现好玩有趣的效果。并且努比亚还对拍摄界面以及交互方式进行重新设计和功能精简，使得操作上手更为简单轻松。

 

 

 


从账面参数就能够看出，努比亚miniS这颗索尼IMX318传感器并不是走高画质的底子，但高达2300万像素使得其有着相当不错的解析力。努比亚miniS有着相当不错的快门速度，即便是在没有光学防抖的加持下。

这就使得其在任何时候都只需要拿起手机按下快门，不用去担心什么握持手势或曝光时间。出片是刷刷的快，在光线充足下足够的画面亮度、艳丽趋向的色彩以及高像素带来的解析力，都足够对得起其身价。

 

 

 



但单个像素面积的缩减以及光学防抖的缺席，都使得努比亚miniS在暗光或夜间环境拍摄时要多加留意，特别是其为了保证拍摄体验，并不会因为环境光线较低延长快门时间。所以其在夜间环境下拍摄，就容易出现画面亮度低、模糊的情况出现。除切换成专业模式进行手动调整，才能够保证画面有足够好的表现。当然嘛，努比亚miniS不还是提供手持曝光这样好用又简单的功能啊。

温而不热的温度、稳而均衡的性能、大而充足的续航

 

 

除外形设计的变化与工艺提升外，努比亚miniS在硬件配置上同样有相当大的改进。其所采用的骁龙625处理器相较上一代所采用的骁龙617处理器，有着更为先进的14nm工艺，使得其能够获得更高的性能，但却有更低的功耗表现。除此之外，努比亚miniS的内存进一步达到4G，并配备有64G或128G储存空间选择，这可一点都不mini呢。

 


当然看到这里，性能控会表示这根本不够看，普通用户则是表示看不懂。确实比起旗舰级的骁龙820/821处理器而言，骁龙625的性能表现要差好几个级别，但其在功耗表现、发热控制以及长时间持续使用时可就要稳定的多。

在实际使用中，骁龙625的性能足够满足系统流畅的运行要求，玩儿简单的如海岛奇兵等游戏自然是不在话下。即便是对性能要求较高的王者荣耀也能够满帧运行，大型团战时虽略有下降但并不会有卡顿或掉帧的感觉出现。等等，这能够说明什么事情啊？

骁龙625的性能并不是亮点，可在14nm的制程工艺以及频率数量都给力的架构设计下，其既能够保证性能需要也提供有漂亮的功耗表现，使得其在长时间持续运作时，并不会有机身严重发热导致的性能削弱的情况出现。打个简单的比喻，骁龙820是短跑比赛刷新纪录的冠军，但骁龙625是能连续跑好几个马拉松的铁人。



不仅是长时间的稳定性能表现，骁龙625同样有着优秀的功耗控制，自然是得益于续航表现。在此基础上，努比亚miniS的内置电池提升到3000mAh的标准线上，搭配独家的NeoPower专利省电算法，进一步的为续航做出贡献。这使得努比亚miniS能够完整的支撑一整天的重度使用，若是作为通话用，好几天忘记充电都不会成问题。

但相对而言，其充电速度可就不大满意。努比亚miniS依然不支持高通快速充电技术，标配的充电器从此前的5V/1.5A提升至5V/2A。虽说其3000mAh电池的充电速度并不会太熬人，但在硬件基础已经支持的情况下，努比亚能够再进一步，这可给使用体验带来不小的提升啊。也只能够期待未来的nubia Z13 mini能够圆梦咯。


作为业界首个实现“全网通”的品牌，努比亚的旗下机型在网络支持这事情上一直是独占鳌头。努比亚miniS的网络通讯部分的功能特性可谓是花多眼乱，你能够想到的以及用不上的都统统支持。诸如三网7模18频、4G+网络以及VoLTE支持，以及副卡支持联通3G网络等。

可也得吐槽啊，努比亚miniS的网络支持是足够的完善，能够随着运营商的一步步推进获得更好的体验。搭配nubia漫游走出国门，也不会成为问题。但在实际使用中，一些信号弱区如楼梯、地下室甚至是室内，都会有通话不清晰甚至是挂断的情况出现。这在别家机型上也会出现的，但努比亚miniS显然还有进步的空间。

 

当然少不了的，还是指纹识别。努比亚miniS将指纹识别传感器设计在机身背面，识别速度与准确度都是常规的快速表现。更为值得注意的是功能方面，除常规的屏幕解锁以及应用加密外，其还能够通过不同场景以及触发事件，分别作为快门以及开启超级截图功能。当然，支持支付宝与微信的指纹支付可是当下的标准啊。

 

是“S”，也是Super，更是Surprise

 


我们完全不意外努比亚miniS的登场，这已经是各家厂商的标准做法。甚至骁龙625处理器、更高像素的后置摄像头、进步点的续航和充电速度，都能够胡说八道对一半。这确实是努比亚miniS的“Super”加强之处，但其在此基础上更进一步。用上的前置1300万像素索尼IMX258传感器、金属一体化机身设计，就可谓是十足的“Surprise”惊喜。

确实努比亚miniS没有旗舰机型那炫酷的无边框设计，也没有光学防抖和快充的加持，更没有C罗同款的心理感知。也确实这些特性使得努比亚miniS并不完美，但依然阻止不了其让消费者喜闻乐见。这就好比飞度雨燕这样的小钢炮，确实没有华丽的内饰或先进的动力，可实在的内部空间、油耗和改造空间，让不少人津津乐道。

甚至让我们有那么点疑惑，努比亚miniS的最初定价会不会是1999元，只是最后定在1599来销售？可以说努比亚miniS既是nubia Z11系列的完整句号，也是nubia Z13系列的提前练兵。
