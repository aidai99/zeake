雷蛇Project Valerie发布：配备三块显示器的便携笔记本

2060 0

是的，你并没有看错，Razer确实将Project Valerie称之为便携笔记本，即便其配备有三块屏幕。Razer Project Valerie最大的特点自然是在于其显示器部分，其采用4K分辨率的17.3英寸IGZO屏幕，并支持NVIDIA G-Sync技术，并且支持100%的Adobe RGB色域，使得其在显示效果与色彩上都有着相当优秀的保证。

但要注意的是，Razer Project Valerie是配备有三块这样的屏幕，使得其能够实现超宽广的视觉效果，以实现NVIAIA Surround沉浸游戏体验。虽说这看起来是挺疯狂的，但Razer一贯不就是如此么？而根据Razer的说法，Razer Project Valerie的定位是工作站，主要是针对工程师、开发人员、设计师等需要移动办公使用。


至于剩下的硬件配置方面，诸如英特尔最新的第七代Core i7处理器，GTX 1080独立显卡等就不必多言。就整体设计而言，Razer Project Valerie可以看成是在Razer Blade Pro的基础上所进行的特别改造，并且即便是有三块屏幕，其机身厚度依然保持在3.8cm，机身重量则是在5.4kg，依然是便携的很。

细节方面，Razer Project Valerie与Razer Blade Pro相同，采用Razer短键程机械按键的键盘设计，配备定制的风扇以保证散热。三块屏幕是通过机械结构以实现，无需任何线材链接即可使用。当然Razer Project Valerie并没有明确的实际售价以及实际销售时间，更多像是Razer的技能炫耀。
