佳能PowerShot G7X Mark II评测：挑战者，参上

39970 4

在1英寸传感器便携相机的领域，索尼黑卡永远是个绕不开的话题和强劲的对手。从初代的RX100发布起到现在RX100的五代同堂，索尼对于有限机身之内极致规格的追求简直令人咋舌。RX100系列下至2000元上至7000元的定价跨度，更是能够吓死一群心理承受能力或是钱包承受能力不足的人们。

难道除了索尼黑卡，我们就没有别的好选择了吗？

当然不是，比如这次的佳能PowerShot G7X Mark II，就是个好选择。


佳能PowerShot G7X Mark II是之前佳能PowerShot G7X的升级款。在这款机器上首发的DIGIC 7影像处理器，让佳能PowerShot G7X Mark II的RAW+JPEG连拍能力提升到8fps；同时DIGIC 7影像处理器也提升了G7X Mark II的追焦能力；影像锐化和高ISO降噪水平也有所提升。甚至从续航能力角度出发，佳能PowerShot G7X Mark II也达到了265张——虽然这个续航能力也不算是特别值得夸耀，但是佳能PowerShot G7X Mark II可以直接使用MicroUSB线缆充电哟。

至于我眼中佳能PowerShot G7X Mark II最突出的东西，那应当是操作性和性价比了。这款目前平均成交价可以下探到3600元左右的小相机，在这个售价水平上提供了相当不错的画质以及非常棒的操作便利性。不论是做备机还是出行的主力拍摄设备，佳能PowerShot G7X Mark II也是没有太多可以挑剔的地方。

简，便


不管佳能的相机怎么被吐槽，EOS单反的操作和使用便利度大抵不会成为槽点之一，PowerShot自然也不会例外。佳能PowerShot G7X Mark II最让人喜欢的也就是这点，除了常规的镜头环、模式/EV拨盘和转轮之外，G7X Mark II还拥有一块可以上下反转的3英寸液晶触摸屏。


这也就是说，在M档（也就是我最常用的档位）下，佳能PowerShot G7X Mark II几乎是可以完全不借助触摸屏进行参数调节的（如果使用自动ISO的话）。即便是想要彻底地控制包括ISO和对焦区域在内的所有参数，G7X Mark II还有一颗可以自定义的按钮，可以分配用于快速调出ISO的调整。至于对焦点和白平衡的选择，用触摸屏也可以轻松愉快的完成。


这还没完，佳能PowerShot G7X Mark II还内建有NFC模块和Wi-Fi模块。支持NFC的智能手机可以和这台相机进行轻触配对，并且直接启动佳能的App（前提是已经安装App）。在手机App里头也可以控制ISO、光圈、快门速度这三个主要参数，虽然不能直接调节白平衡设定有些遗憾但有这三个参数倒也是够用了。至于从相机上直接下载图片，自然也是必备功能之一。




值得表扬的，佳能PowerShot G7X Mark II的机身握持感其实是相当不错。但由于G7X Mark II因为机身过于小巧，在单手握持状态下其实是相当不方便对各种参数进行调节的。若是大部分时间喜欢单手持机，就老老实实用G7X Mark II的自动挡吧。

摄，影


佳能PowerShot G7X Mark II用的是一颗2000万像素的1英寸背照式CMOS传感器。搭配的镜头等效焦距为24-100mm，提供F1.8-F2.8的最大光圈，支持防抖，NTSC制式下最多可以达到1080p@60fps的录影能力。













（以上样张为M档JPEG直出）

从M档的拍摄体验说起来，佳能PowerShot G7X Mark II表现其实算是相当不错。首先G7X Mark II的防抖能力相当不错，在手持状态下可以比较轻松的使用1/30秒的快门，所以在拍摄静物时自然就可以使用更低的感光度以换取更纯净的画面。

其次，佳能PowerShot G7X Mark II的镜头即便是在光圈全开的情况下，也拥有相当优秀的锐度表现。所以在拍摄静物或是人像的时候，便可以放心大胆地将光圈全开，也不用担心中心锐度会不足。


在拍摄静态照片时，G7X Mark II上最影响拍摄体验的问题其实还是出在对焦上面。单纯从每次的AF准确度和速度来说，G7X Mark II还算是不错，不过G7X Mark II有些时候会难以捕捉细小的对焦主题（比如树枝）。

并且退一步说，G7X Mark II的手动对焦功能给人的体验也相当糟糕。需要半按快门的同时扭动镜头环对焦距进行细微的调节不说，镜头环对于焦距的调节的调整速度和灵敏度都极其难以把握，以至于到最后我都根本不想用手动辅助G7X Mark II的对焦。若不是自动AF的表现还算不错，这台相机怕是要在我这拿个负分滚粗的评价。







（以上为全自动档位直出jpg样张）

对于那些喜欢自动挡走天下，抬手随便拍的人们来说，G7X Mark II的表现是很不错的。在自动挡之中，G7X Mark II会倾向于以较快的快门速度保证细节清晰。曝光度、色彩以及宽容度也没有太多可以挑剔的地方。

凭借全新的DIGIC 7影像处理器，G7X Mark II的高ISO直出JPG也有很出色的细节表现。不过在ISO3200之后，G7X Mark II的JPEG就会有比较明显的油画感，再往上的话就更多是“拍个亮”的画面细节品质了。









（以上为M档拍摄，Lightroom/Photoshop后期调整处理样张）

若是愿意使用Lightroom这类软件稍微修修图，那么G7X Mark II是绝对能够给到你惊喜的。G7X Mark II的传感器宽容度相当不错，所以其所输出的RAW也有非常大的可调节空间。

对于一款不到4000元就能买到的紧凑型相机来说，我们对视频录制其实是并不抱太多期待。但稍微使用G7X Mark II录制了一些视频之后，我们反倒感觉它有些让我们惊讶了。







（以上为1080p分辨率，60fps的录制视频截图）

G7X Mark II的视频录制规格并不算突出，全高清分辨率加上最高60fps的帧速自然是能让人感觉“够用”。在所有帧率下所获得的全高清视频所呈现的画面细节处于一个合格水平，反倒是色彩表现让人相当满意。另外在广角端录制视频时，G7X Mark II的光学防抖也有还算不错的表现，只不过长焦录制视频时，手部的晃动就会对画面造成比较明显的影响了。

所以总的来说，G7X Mark II的视频录制功能其实最多还是限于日常的生活记录以及Vlog使用——毕竟一台不到四千元的机器，若是想要它输出品质极为出色的4K视频，也真是强人所难了。

挑战者，参上


在过去，诸如佳能PowerShot G7X Mark II这样的1英寸“小”机器大多是被当做摄影爱好者们在不想背着他们的大单反时的一个替代品。不过随着传感器、镜头和影像处理器的进步，1英寸的小机器们也能够拥有不俗的性能。

也就是说，曾经的备胎如今已经拥有了入主正室的实力。

这看上去像是卡片机们的自我修养，但实际上这也是卡片机们在被智能手机抢走市场之后所必然的发展方向。索尼的黑卡系列无疑是这一领域绝对的霸主，在数码单反界叱咤风云的佳能，反倒是1英寸卡片机领域中的后生以及挑战者。


既然是所谓挑战者，那么G7X Mark II就一定得有一些过人之处。在同等的预算之中，G7X Mark II不但拥有十分出色的画质和色彩表现，其操作和连接性上的表现更是让人满意，大光圈的出色虚化效果也极为诱人。加上十分实惠的价钱，G7X Mark II是一位合格的，并且具有很高竞争力的挑战者。

会有不少人说佳能PowerShot G7X Mark II没有内置EVF和热靴是个遗憾，但我却并不这么觉得。在实际使用之中，EVF只有在非常强的阳光下才会拥有真正的实用价值，热靴的必要度和实用性自然会更低。


那么佳能PowerShot G7X Mark II真正的缺点何在？总的来说便是十分鸡肋的手动对焦功能、以及屏幕在强烈环境光线下表现一般这两点。但考虑到G7X Mark II十分实惠的售价，这两点我觉得是可以接受的啦。

那佳能PowerShot G7X Mark II值得买吗？反正自从我收到G7X Mark II之后，确实很少再去使用之前我用来工作的EOS 60D+适马18-35了。不知道这样能不能表明我对这台小相机的态度呢？
