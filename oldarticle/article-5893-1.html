HIFIMAN Edition X V2评测： 轻音体柔易推好

65092 9

在很多人的印象里，大耳所代表的就是笨重的台机，复杂的系统调教，还有令人绝望的线材和电源处理，似乎唯有这样整套台机系统的驱动下，才能去挖掘出一副大耳所拥有的全部潜力。当然也不是大耳随身不能听，比如 Beyerdynamic T1 或是 Sennheiser HD800，尽管本身阻抗就高达 600Ω ，但实际手机驱动并不会可怕到难以出声，区别在于这样的表现那是相当对不起耳机本身啊。

近些年火热的随身市场在一定程度上带动传统 HiFi 器材的火热，很多对随身玩腻味的用户对大耳也有想要尝试的冲动。厂商们自然会推出类似 Philips Fidelio X2 等一系列低阻大耳，受到不少用户的追捧，随身设备并没有因为改听大耳而遭到闲置。但这样的大耳是很难达到台机驱动下大气宽容的表现，于是乎 HIFIMAN 站了出来，针对动圈大耳难以回避的驱动问题进行解决，换用更加好驱动的平板振膜研发易推大耳，这就是 HIFIMAN Edition X V2 。

细致入微的精益求精


相比国际大厂 Sennheiser 传统的蓝黑色外包装设计风格， HIFIMAN 非常简约的采用黑底加耳机线稿的纸套设计。背面清楚写道耳机的具体参数，其高达 8Hz~50kHz 的回放频率完全超越人耳收听范围，而仅仅 25±3Ω 的阻抗和 103dB 的灵敏度更是一目了然，这是为了随身而设计的大耳。



作为一款售价近乎万元级的产品，采用简单的纸盒包装相信厂商自己都不会放过自己。外包装的纸盒下，包裹着一个尺寸和外纸套尺寸相当的收纳皮箱。皮箱触感细腻，还有提拉带方便用户开箱，里面体积巨大的海绵包裹住耳机提供完美的保护。唯一感到有点遗憾，并没有配上一个类似 AKG K702 的展示盒，平常要是随手放置耳机塞皮箱不咋方便，建议用户日后购买个耳机架收纳。






皮箱中间部分放置耳机线材和配件，本身 HIFIMANEdition X V2 就是一款易推型大耳，采用的是双边可换线设计，自然在线材上提供了多种聆听方式的选择。HIFIMANEdition X V2 提供一条 1.5m 长使用 3.5mm 欧亚德弯插耳机线一条；一条 3m 长配备采用纽崔克 6.3mm 插头的台式耳机线；一个 6.5mm 转 3.5mm 转换头。

线材手感柔软细腻，线材外保护层上有防缠绕设计，同时选用透明的材质让用户可以一目了然线材用料咋样。 HIFIMAN Edition X V2 搭配的两条线材，均是采用铜银混编线基，以保证高质量的声音回放。


HIFIMAN Edition X V2 使用的是平板振膜单元的耳机，自然在腔体设计上和传统动圈耳机不同。一般高端动圈大耳都会针对人耳的自然倾斜角度，对动圈单元采用斜置设计。而平板振膜耳机受困于本身单元体积过大，难以斜置单元；于是才耳机罩上采用3D剪裁设计——前薄后厚，充分包裹耳朵的同时，给予平板单元带来最佳的回放位置。


HIFIMAN Edition X V2 的头梁调节非常直观，直接在头梁上进行标注，段位清晰明了，可调整范围大。头带具有相当不错的透气性，减缓长时间搭配对头部的压迫感。实际佩戴起来，由于 HIFIMAN Edition X V2 重量基本上集成在单元两端，对头部负担不大，长时间佩戴只是稍感闷热而已。


对于平板振膜耳机而言，大多数人对其印象，可能是Audeze品牌的一系列平板大耳上。其旗下的平板大耳以重量笨重而著称，用户普遍吐槽要想听平板耳机得先练习一下脖子的支撑力。 HIFIMAN Edition X V2 作为一款易推到底的大耳，自然在重量上要做出巨大的削减。

传统动圈旗舰大耳例如 Sennheiser HD800 重量仅 330g ，而平板大耳 Audeze LCD-2 重量高达537g，这样的重量是难以接受的。但 HIFIMAN Edition X V2 硬是活生生把重量削减到 399g ，可以说也就是稍微比普通动圈大耳重了点，比起 Audeze LCD-2之流简直轻盈的能飞起来。

脱离随身领域的台机声


谈及 HIFIMAN Edition X V2 具体声音这块，笔者实在很是羞愧。本人手上驱动 Sennheiser HD600 换线 CARDAS 的系统难以把HIFIMAN Edition X V2 的潜力充分挖掘出来，仅能勉强喂饱其信息量的需求，而耳放驱动这块实在是无能为力。 HIFIMAN Edition X V2  是款需要高电流耳放驱动的耳机，如果用户有上台需要可以试试相关产品。

值得注意的是， HIFIMAN Edition X V2 的说明书标注本款耳机不能使用超过 2W 的耳放进行驱动，否则会出现过推现象。所以以下听音评价都由 HIFIMAN 出品的 SuperMini 和 MegaMini 驱动下的出，毕竟这是一款针对随身所设计的旗舰大耳嘛。


HIFIMAN Edition X V2 开声就给人很惊艳的感觉，甚至可以说完全和市面主流的平板耳机调音走向完全不同。相比 Audeze 出品一系列平板大耳温暖的中频质感， HIFIMAN Edition X V2 直接玩起了追求中频线条感和高频透明度。可以说这和传统平板大耳完全对立，甚至和 HIFIMAN 一直强调的“松、润、甜”完全相悖。但细细品味起来，追求这样的调音有其合理之处。

如果学习传统平板大耳追求中频厚度和味道的话，受限于前端器材的揭示力不足和驱动力较低，反而给人糊成一团的难听的印象；但走现在这种追求线条感和透明度的路线，更加容易凸显前端强大的揭示力，展现平板耳机独特的魅力。


HIFIMAN Edition X V2 作为一个专门为随身设备所开发的旗舰平板大耳，自然对各类随身设备有着超乎寻常的适应性。别的不说，仅仅是声音通透的质感上，连 iPhone 都能驱动的很明显，可以想象对前端要求的真的不高。


当然要想在随身上体会到 HIFIMANEdition X V2 类似台机的那种宽松大气的听感，自然要给它来几个不错的 HiFi 播放器才行。选用 SuperMini 和 MegaMini 进行分别驱动，信息量和耳放驱动力上无疑是 SuperMini 更加有力，声音凝聚感更强，甚至低频都有点类似台机驱动的动态起伏。

惊喜在于这样简单的搭配下，对声音的纵向的表现也有着超乎意料的表现，实在是惊喜连连。但这样的惊喜换到 MegaMini 驱动上就有点失色，声音不如 SuperMini 结实，唯有人声这块透明质感相比 SuperMini 驱动还来更加明显些摆了，看来 HIFIMAN Edition X V2 想要驱动的好，还是建议用户尝试搭配一些随身耳放进行尝试。


针对 HIFIMAN Edition X V2 的随身器材推荐，如果你不喜欢捆绑设备的话。那么乾龙盛 QA360 MOD 版、乐彼金 LP5 或是旷世 QP1R等耳放部分比较强悍的前端可以纳入考虑，当然更高端的 AK380 AMP 套装或是金菊花都可以考虑。 HIFIMAN Edition X V2 虽然限定不能超过 2W 的耳放驱动功率，但这样的指标在随身设备上不是这么容易超越，所以放心的选购吧。

而针对 HIFIMAN Edition X V2 的台机系列搭配的话，针对其声音透明度质感较高的特色，解码部分比较推荐类似 Prism Sound Callia Lyra2 这样强调声音平衡性的解码。而针对其耳放部分要解决 HIFIMAN Edition X V2 人声有透明感却没足够的感情问题，同时还要考虑耳放输出功率不超过2W的要求，推荐一些味道胆放进行驱动，这方面个人口味不一，笔者就不推荐啦。

快餐与正餐的抉择


HIFIMAN Edition X V2 在笔者看来更像是一款针对“小白”台机用户选购的第一款大耳产品，它能够成为对随身感到厌倦，但又不想完全抛弃随身设备的一款替代品。尽管 HIFIMAN Edition X V2 直推的声音能完全脱离随身领域的范畴，但致命的问题是组建台机系统时耳放不好搭配。

不过在如今的时代，不求产品面面俱到，只为能在特有环境下发挥到极致即可。从这方面来看， HIFIMAN Edition X V2 无疑开启了一个随身的全新篇章：不同于传统平板耳机的调音、精益求精的耳机工艺、令人惊叹的重量体积、还有对驱动设备的低廉要求。


不过从大耳的完整性上来说， HIFIMANEdition X V2 无疑是不健全的。本身的平板应该拥有的潜力被迫让位于易推性上，一旦正儿八经上台机和旗舰们 PK 时，非常容易暴露出底气不足的劣势。也就正如日常吃饭一般，你要图快省心，那当然麦当劳之流的洋快餐简单粗暴；但要吃的舒心，还是老老实实大米饭来的靠谱哇。
