AKG N40评测：圈铁协奏曲

18832 0
[attach]261958[/attach]

从森海 IE80 到双进气孔的 IE800; 舒尔 SE535 到四单元动铁 SE846; 威士顿 W4R 再到六单元动铁 W60, 诸位烧友耳熟能详的品牌对应市场给出了 "青出于蓝" 的耳塞作品, 唯独 AKG 让人比较意外: 在当年的两铁一圈 K3003 价格 "拦腰斩" 之后, 时隔五年, 又推出了定位稍低一档的 一圈一铁 - N40.

这其中 AKG 被 哈曼卡顿 收购, 再到业界传出三星重金吞并哈曼卡顿的新闻, 五六年时光, 仿佛弹指一挥间.

[attach]261959[/attach]

一如既往 AKG N40 为橱窗式展示包装, 从背面即可看到耳塞.

[attach]261960[/attach]

产品参数

频响范围: 10~40000Hz
灵敏度: 115dB SPL/V @1kHz
最大输入功率: 10mW
阻抗: 20Ω
线长: 1.2m
插针接口: MMCX
插头规格: 3.5mm
重量: 22g
配件: 3 对声音调节滤网, 4 副硅胶套 ( XS,S,M,L ), 航空转接头, 清洁棒, 1.2m 线材, 1.2m 带线控线材 及 收纳包

官方参数页面 - http://eu.akg.com/N40SIL.html

1.MMCX 插针可兼容 宜丽客 EHP-CB1000 以及 松下 RP-HDE10 等耳塞.

2.官网提供的参数稍显 "含糊", 在反复查阅资料后, 确认 AKG N40 为一个平衡电枢式 ( 动铁 ) 单元 + 一个动圈单元 结构. ( K3003 为两铁一圈 )

[attach]261961[/attach]

抽出最外层包装后, 内层如书本翻开: 由小到大 4 对硅胶套 及 耳塞清理工具.

[attach]261962[/attach]

航空转接头及 3 对 声音调节滤网 ( 逆时针方向扭下 ): Bass Boost "低音增强", ( AKG 标榜的 ) Reference Sound "参考及声音" 与 High Boost "增强高音". - 从调节进气孔而改变低频量感的 森海 IE80, 双进气孔双层滤网的 IE800, 再到同样皆可更换滤网的 K3003 与 N40, 异曲同工.

[attach]261963[/attach]

N40 与 威士顿 W4R 佩戴方式相同: 必须绕耳佩戴. 其线材可拔插更换, 插针接口为 MMCX, 自带线材 最外层为尼龙编织 ( Y型导线分割部分未覆盖 ), 有点象大家熟悉的游戏鼠标线材. 红色 "R" 为 右, 蓝色 "L" 为 左侧, 右侧线材上设有盲点.

[attach]261964[/attach]

N40 自带两条线材: 其中一条含有线控与麦克风.

[attach]261965[/attach]

经测试其线控可同时支持 IOS 与 安卓手机, 平板 等设备 暂停 / 播放音乐, ( 但无法操控本测评中的音乐播放器 ) 麦克风及其开关位于线控背面.

注:

测试以 FiiO 飞傲 x7 ( 固件版本 FW 3.1.4 Beta, 安卓 5.1.1 系统 ), 爱欧迪 Plenue M ( 固件版本 1.18, 开启耳机模式, 并设置到最大音量 140 可作为线性输出, JetEffect 7: Normal ), Cayin 凯音 i5 ( 固件版本 2.0, 安卓 4.4 系统 ) SONY PCM-D50 作为前端, 采用线性输出轮流搭配 奥莱尔 Aune B1 耳放进行, 连接线材为飞傲 L16 对录线. 森海塞尔 IE80 与 AKG N40 均选择最小号硅胶套, N40 选择不含线控线材 及 Reference Sound 滤网. 凯音 i5, 奥莱尔 B1, AKG N40 煲机均超过 200 小时以上, 爱欧迪 Plenue M, 飞傲 x7, SONY PCM-D50 与 森海塞尔 IE80 购置皆超过 1 年. 为方便网友扼要了解, 在测评中顺带附上低码率 MP3 试听地址.

Cayin 凯音 i5 + Aune B1:

Aune B1 耳放上并没有明显的刻度标记: 低增益模式, Class A 位于减号处, 大约 40% 音量驱动 AKG N40. Ingram Washington 的声音突出且细腻, 萨克斯与钢琴的伴奏凸显着爵士乐的 '慵懒' - 宛若漫不经心的情调, 3 分钟的曲目让人回味无穷.

换回 IE80, 125dB / 16Ω 阻抗的 森海 明显易推不少. IE80 男声上的染色比较明显: Ingram Washington 的嗓音更显厚实, 富有磁性, 但解析力上这条动圈老将明显逊色 AKG N40 几分, 乐器方面 IE80 下的萨克斯更为突出.

FiiO 飞傲 x7 + Aune B1:

网友 "遥远的加州" 点评令人印象深刻 - "其实, 这部电影, 最令我感动的是最后的镜头: 100 岁的 Rose, 躺在床上安静的睡着了. 而床头前, 放满一张张照片, 有在非洲骑大象的, 有戴着飞行帽与飞机合影的, 有和她的孩子们合照的, 有去夏威夷冲浪的. 这是 Rose 生命中没有 Jack 的时光, 但却是 Jack 在即将沉入海底前托付 Rose 的话, 她用一生去做到了."

当爱尔兰哨笛响起, 熟悉的旋律在耳边萦绕: Celine Dion 的声音甜美, 沁人心脾... 稍让我意外的是 N40 横向声场比较出色, 音乐进入高潮, Celine Dion 宽广的音域, 不禁让我感慨这 200 多小时的煲机没有白费. N40 在女声方面的造诣, 保持了 AKG 一贯的水准.

IE80 下的《My Heart Will Go On》明显逊色几分: 无论是开场乐器的中高频延伸, 还是女声部分. 森海的女声少了几分细腻, 不够抓耳, 但比起直推, IE80 的表现还是有所进步.

SONY PCM-D50 + Aune B1:

D50 与 B1 的搭配略显 "温吞", 尽管 Aune B1 磨平了其 "干, 冷, 硬" 的棱角, 但在与 N40 的搭配中, AKG 出色的高频延伸也丢了影子.《HANA-BI》迂回沉郁的曲调, 仿佛在上一套组合中另人印象深刻的风格, 如转瞬即逝的烟火. ( "HANA-BI" )

提起 久石让, 很多人首先想到的是他与宫崎骏的作品, 但这张其与北野武的杰作, 同样扣人心弦. IE80 下的钢琴声更为 "沉重", 泛音也更丰富, 在这样一首乐器构成简单的曲目里, IE80 的表现稍占上风一点, 也许本是 IE80 与 D50 相称的搭配, 亦或是电影《HANA-BI》惨重的主题.

ラブストリ—は突然に ; 虾米音乐

[ APE / 914Kbps / 44.1KHz / 16bit ]

Cowon Plenue M + Aune B1:

电子乐 与 "花哨的音乐后期", 它们大概是我印象中早期的日本流行乐. 即使关闭所有音效, 爱欧迪 PM 听起来仍是一款音染较多的播放器, 这样的前端组合与《东京爱情故事》中的主题曲《突如其来的爱情》倒算是来场 "不期而遇". 我个人比较顾虑的因器材解析力出色, 而导致搭配听起来腻味或刺耳的情况并未出现, ( 一部分日本流行乐本身后期过于夸张 ) 但整曲用 N40 听起来也并不出彩.

由于 IE80 偏下盘的缘故, 整曲听起来乐器近了不少, 那个年代的录音技术无法与今日同日而语, 即使在更擅长男声的 IE80 下 小田和正的声音听起来也并不抓耳.

抛开器材之间的对比,《突如其来的爱情》一曲更重要的意义是: 每当旋律响起, 这部电视剧的情节历历在目.

-----------------------------------------------------------------------------------------

总结

20Ω 阻抗 / 115dB 灵敏度的 AKG N40 乍看是一款易推的耳塞, 但在尝试与手机搭配后, 我觉得其声音几乎可以用 "平淡无奇" 来形容, 一如既往, 它和 K3003, K701 等耳机特性相仿, 对前端有一定要求.

在反复尝试多款播放器以后, 搭配合理的前提下, 我认为 N40 仍保持了 AKG 擅长女声, 弦乐, 小编制的特点, 比起 "染色较多" 的爱欧迪 Plenue M, 它更适合音染相对更少的 凯音 i5 或者 飞傲 x7 等 '国砖'.

从早期的 奥莱尔 T1 电子管, 解码耳放一体机 奥莱尔 X1, 僵尸 ECCI ST6MK2, 再到今天的 奥莱尔 Aune B1, 国产耳放的进步确实可圈可点, 不过其产品外形, 工业设计, 我觉得仍有 "进步空间".

也许 '圈铁' 早已不再是 hifi 圈里的热门话题, 与 威士顿 / 舒尔 相仿的绕耳式结构 ( 长时间佩戴没有明显不适感 ), 延续自家的 K3003 可更换滤网调节声音风格的设计, 稍许迎合市场做出的双线可换, 一条包含智能线控的改动... ( 不再象 K3003 推出 K3003i 版, N40 智能线控可同时支持 IOS 与 安卓手机, 平板 等设备 )

如果将 K3003 比作单反中的 佳能 5D Mark 3, 或许 N40 正是市场期待的 6D Mark 2, 一款博采众长的产品.
