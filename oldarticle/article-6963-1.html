努比亚Z17上手玩：存异于大同

262323 9

全球首款骁龙835+8GB RAM、全系标配UFS 2.1、2300万像素+1200万像素变焦双摄、高通QuickCharge 4+……努比亚Z17的发布会真是让人看着心潮澎湃。2799元的标准版就能够给人非常体面的配置和外观，至于3999元的尊享版，则可以成为你向朋友们炫耀的资本。

这还没完，正当我们以为努比亚Z17只是参数党的又一次狂欢的时候，努比亚还拿出了“玻璃金属”这一概念。努比亚Z17极光蓝版本将玻璃与光影嬉戏的本领附着在扎实可靠的金属机身上面，让我们惊叹：原来金属机身也可以做出这种质感。


好吧我承认，从努比亚Z9开始直到今天和我们见面的努比亚Z17，努比亚旗舰手机的ID设计语言已经原地踏步了3年。Z9亮相当时是一款很惊艳的产品，Z11将无边框的设计语言进一步成熟化，Z17大概就是这种设计最终极的进化。

这种思路没有毛病，只不过上一个把同一个设计语言用了3年的公司（那个公司可是用同一种设计语言出了6代旗舰手机哟），可是切切实实地从全球市场份额前五跌倒了Others里头啊。

努比亚Z17漂亮么？确实漂亮，只不过在这个智能手机沦为快消品的时代，努比亚可能要在下一代旗舰产品上加入一点新的设计元素来免除我们的审美疲劳咯。


努比亚Z17的外观细节和机身其实还是有些值得说道的，首先便是所谓的“单缝天线”机身。从整体的外观来看，努比亚Z17的后背上确实只有唯一一条天线隔断。但是若将努比亚Z17抓在手中仔细观摩细节，便会发现努比亚其实是将机身上部的天线隐藏在了机身上部的边框之中。在我们黑色的评测样机上面极其不明显，但若换做是红色或是金色款，应当还是很容易被看出的。

其次，便是努比亚Z17这次也毅然决然地加入了消灭3.5mm耳机口的大军之中。全机身上下唯一的接口便是位于底部的USB Type-C，包装盒中附赠的Type-C转3.5mm耳机接口线自然也不出人意料。


最后，便是努比亚Z17全新加入的户外防水和生活防水功能。从严格意义上来说，努比亚Z17的防水功能应当只能够对抗一些轻度的液体溅洒，而非如同IP68的三星Galaxy S8一般全身下水之后还能镇定自若地谈笑风生。但从我实际作死的结果来看，努比亚Z17还是能够稍微泡一泡水的。

哦，另外有一点是努比亚Z17用户们一定要注意的。那便是努比亚Z17在下水过后，是绝对不能够马上插上USB线缆充电或是传输数据。因为即便是机身表面看起来已经干燥如初，其USB接口之中可能还会含有相当量的水分。此时若是进行充电等操作，便会有引发短路之风险。


努比亚Z17在硬件配置上可谓是极其激进的。除了2017年旗舰手机标配骁龙835处理器之外，努比亚Z17在RAM配置上也是极其慷慨大方。全系6GB RAM起跳8GB RAM封顶，让这款无边框旗舰在性能这方面有着绝对的统治力。





另外值得一提的，努比亚Z17是骁龙835处理器还是目前市面上唯一的“真·满血版”骁龙835。比起小米6、三星Galaxy S8、索尼Xperia XZ Premium只能飚到2.36GHz的骁龙835，努比亚Z17的骁龙835最大频率能够达到2.45GHz。额外的一点性能，使得努比亚Z17的安兔兔随手一跑便是17.9万分的成绩，硬是压过了小米6一头，只不过优势不大而已。

在这里或许有人就该说了，“手机的整体体验并不由单一硬件的性能决定”，或是“这点频率的提升充其量也就是让跑分更好看而已”，以及“跑分不绝对代表日常使用体验”云云。


没错，我同意。但是“我手机跑分比你高”这种东西，说起来就是那么让人舒爽啊。

所谓“经济基础决定上层建筑”，有了骁龙835、6GB LPDDR4X RAM和UFS 2.1这样的顶级“经济基础”，那么nubia UI 5.0这个“上层建筑”自然就可以建的气势恢宏。



nubia UI 5.0咋看之下其实和4.x版本没什么区别，但实际上改动的地方其实非常多。除了一些动态图标、小部件以及UI细节的修改之外，功能性操作也和前作略有不同。

举例来说，我在拿着努比亚Z17的前5分钟内，我都没找到Wi-Fi网络的快捷开关在哪儿，只能一脸懵逼地对着空荡荡的下拉通知栏徒劳地左右滑动。只有在看到旁边老司机的操作之后，才发现nubia UI 5.0之中将快捷控制全部放在了从屏幕底部呼出的上拉栏之中。



除此之外，nubia UI 5.0里头还使用Fitcard代替了Z11上面使用频率并不高的“边框内划展示桌面”功能。Fitcard的本质其实相当于不少国产UI中的“负一屏”，用于展示提醒、手机状态等等信息。

从整体的功能性上来说其实Fitcard并不算十分丰富，但当做一些诸如清理内存、开启省电模式等等的日常操作快捷方式，倒也是挺好的。



最后一个大改动，便是nubia UI 5.0中加入了桌面长按图标弹出快捷方式的功能。其功能性和iPhone的3D Touch类似，只不过努比亚Z17上面只是单纯地长按就会弹出。

这一功能在系统的内置App上面运行得挺不错的，但是对于第三方App，则需要给对应的开发者一点时间来适配。至少在目前，诸如微信、QQ这样的App都还没有对其提供应有的支持。


最后，也是最重要的，便是努比亚Z17的相机模组了。

努比亚Z17的后置相机模组在我看来堪称是“奢华”——其1200万像素的主相机采用了和三星Galaxy S7、S8系列一样的1/2.55英寸的全像素双核对焦感光元件；副相机的这颗2300万像素感光元件则是Z11mini S同款，另外还通过焦距设置达成2倍光学变焦。

当然，在光线不足的情况下，努比亚Z17会自动屏蔽掉那颗2300万像素的相机。此时即便是开启2倍变焦，所得到的画面也只是那颗1200万像素传感器数码变焦的结果，这点和iPhone 7 Plus并无二样。

     









也许是用了类似传感器的原因，努比亚Z17的初步成像风格给我的感受其实和三星Galaxy S7十分相似——只不过努比亚Z17对于细节的处理手法略有不同。

努比亚Z17的1200万像素主相机对于细节的锐化程度并没有Galaxy S7那么高，所以画面看起来也会更加自然亲切些，不会有Galaxy S7那种咄咄逼人的画面锐度（实际上二者的画面信息量并没有太多差别）。


硬要说有什么遗憾的，便是努比亚Z17上取消了光学防抖镜头了。因为光学防抖镜组的缺席，努比亚Z17在极暗光环境之中就需要单纯依靠传感器感光性能和内置降噪算法来获得清晰的影像。但即便如此，努比亚Z17的表现也十分让我满意——至少在已经发布的同价位的手机之中，努比亚Z17可以将竞争对手们挨个按在地上反复摩擦。

总的来说，努比亚Z17给我的第一印象其实有点复杂。不管是外观设计还是系统UI，努比亚Z17给我的区别和努比亚Z11小得几乎可以忽略。但也正是诸多细节的不断累积，使得努比亚Z17和前代产品相比产生了质的变化。


精致、漂亮、帅气并且性价比非常凸出，这些曾经被我用来形容努比亚Z11的词语，放在努比亚Z17上面依旧是成立的。也许努比亚已经将一个设计语言沿用了3代旗舰产品，但每一年努比亚都在努力地让这种设计语言变得更成熟细腻。以恶俗点的话语来说，这便可称作是一种“工匠精神”了。

但话说回来了，不知道在下一代的努比亚旗舰机上，无边框又会有什么样的进化呢？
