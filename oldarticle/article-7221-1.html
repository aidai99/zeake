更高效的生产力 商务本推荐

83082 5
在如今用户多样化需求的刺激下，PC厂商开始根据不同定位推出各种种类的笔记本。有极致性能的游戏本、满足家庭使用的全能本、当然还有针对办公人士的商务本等。对于初次选购笔记本的“萌新”来说，这么多种类的笔记本，选择起来肯定是相当困惑。

就拿商务本来说，理想型商务本应当兼顾充沛的性能、素质良好的屏幕、尽可能的轻负担、还有较为充裕的接口数量，这样才能满足出行办公的要求。以下就推荐几款商务本新秀，如果你对选购商务本感到迷茫的话，相信本文能够给你一点帮助。

小米笔记本Air 这次我们谈综合性价比


小米作为笔记本市场的新人，却交出了一份相当漂亮的答卷。小米笔记本Air按照屏幕尺寸分为12.5英寸和13.3英寸两种版本，前者主打长续航办公，后者主推轻薄高性能。


12.5英寸的版本可选IntelCore m3-7Y30和i5-7Y54两款处理器，并内置HDGraphics 615集成显卡。存储空间最高为256GB SSD，内存则提供了4GB和8GB。13.3英寸版本配置全面升级到Intel Core i5-7200U和Intel Core i7-7500U，并采用NVIDIA GeForce MX150 独立显卡，满足你对性能的需求。

小米笔记本均搭载了1080P分辨率的屏幕，并搭载了Type-C接口、USB3.0、HDMI和3.5mm耳机孔，其中13.3英寸的版本还额外多了一个USB3.0接口。


如果按照配置性价比来看待的话，这两款笔记本并不占优。但笔记本可不仅仅是看配置和价格，它需要综合做工、便携性、拓展性等多方面考量。这点上，小米笔记本属于尽管你能在某个方面打败我，但我能在其他方面找回场子。比如，小米笔记本的金属机身、较好的屏幕素质和良好的拓展性，在同价位笔记本中可是较为少见的。

那么小米笔记本适合什么人呢？首先你想要笔记本轻薄，其次还得考虑到一定的游戏性能。如果是按照这样次序的需求，那么13.3英寸版相当适合你，而12.5英寸版就只适合一些追求笔记本轻薄便携的用户。尽管12.5英寸版本的配置较弱，但应付起办公需求还是可以接受的。

惠普Pavilion x360 13-u140TU 低价也能翻转屏


如果你预算只有4000到5000左右，还想买一款轻薄型的翻转屏商务本，那么惠普Pavilion x360能够很好的满足你。这台笔记本是该价位少有的带屏幕翻转功能的产品，其搭载的13.3英寸高素质的触控屏绝对能讨好你的眼睛。


这台笔记本配置方面比较保守，对比同价位轻薄本普遍搭载的SSD，惠普居然给出了1TB机械硬盘的选项。另外，惠普Pavilion x360的厚度接近20mm，净重1.58kg，这样的体积表现作为一款轻薄本多少有点说不过去。另外，你要是购买了惠普Pavilionx360的机械硬盘版本就没有高速的M.2接口，所以这里还是推荐小伙伴尽量购买SSD版本。


接口方面，惠普Pavilion x360提供了一个USB2.0和两个USB3.0接口、HDMI和3.5mm耳机孔。配置方面，可选Intel Core i5-7200U和Intel Core i7-7500U，内存提供了4GB到8GB，存储空间可选1TB硬盘和最高256GBSSD。

对于商旅人士来说，续航时间长短是能不能保持办公效率的关键。这里，惠普Pavilion x360内置了3芯锂电池，可以提供长达9小时的续航时间。在面对不方便使用笔记本形态的情况下，可翻转屏笔记本明显要来的更有效率。惠普Pavilion x360非常适合那些预算吃紧，又想体验翻转屏笔记本的朋友。

三星900X3M 超轻薄的商务本


如今的PC厂商们为了让产品更加有差异化也是蛮拼的。为了让笔记本更加浅纤薄，苹果率先将常见的接口全部替换成Type-C接口，甚至还将笔记本键盘的键程也进行了阉割。因为只要将笔记本做薄，就能在第一眼上吸引你的目光，顺带还能展现厂商的设计实力。

可是，对于主打便携性的商务本而言，最关键的衡量目标并不是厚度。而应该是在合理厚度和适当接口数量上，做到尽可能的轻便。这里就要推荐的就是一款主打重量轻盈的商务本三星900X3M。


三星这款商务本提供了13.3英寸和15英寸两个版本，这里主要讲13.3英寸的版本。其配置方面提供了Intel Core i5-7200U和Intel Core i7-7500U，内存提供了4GB到8GB，存储空间最高可选512GBSSD。比较特别的是，采用了1080P 72%NTSC色域的PLS屏幕，支持180°开合。


回到重量上面，三星900X3M在采用全金属机身的前提下，将裸机重量压制在840g，厚度为13.4mm。别看机身这么轻薄，在笔记本接口上并没有进行缩减，提供了两个USB3.0接口、3.5mm耳机孔、MicroHDMI和Mini VGA，能满足基本的日常办公需求。

当然这样一款轻薄极致的笔记本，还是有不少并不合理的地方。摸不着头脑的是，为何没有搭载上时下流行的Type-C接口，反而放上小众的Micro HDMI和Mini VGA接口？抛开接口上的特殊来看，三星900X3M凭借着自身轻薄的特性，足够差旅人士对轻负担出行的需求。

华硕灵耀3 Deluxe 独树一帜的外观配色


厌倦商务本都是一成不变的灰白黑配色？那么华硕新出的灵耀3 Deluxe绝对能让你眼前一亮。

华硕灵耀3 Deluxe提供了Intel Core i5-7200U和Intel Core i7-7500U处理器可选，内存可选配8GB和16GB，储存空间最高可选择512GBSSD。使用了一块14英寸1080P分辨率的72%NTSC色域的IPS屏幕，拥有高达84%的屏占比空间。整机厚度约为12.9mm，重量约为1.1kg。


相信仅仅是看看华硕灵耀3 Deluxe的硬件配置并没有什么值得惊奇的地方，但这款笔记本最让人惊艳的就是它独一无二的配色。深蓝色的华硕同心圆设计金属拉丝A面，再加上骚气十足的金色包边设计。另外，C面的键帽设计也与A面相同。无论你喜欢不喜欢这样的配色，不得不承认这样的设计相当精致。


与华硕灵耀3 Deluxe外观配色成正比的，还有它高昂的售价。在售价近乎W元的级别，你的竞争对手可不再是戴尔XPS 13这样的选手，而是微软Surface Laptop和ThinkPad X1 Carbon。无论是品牌号召力，还是产品特性来说，华硕灵耀3 Deluxe并不具备和这两位大佬直接竞争的实力。如果你对华硕灵耀3 Deluxe的配色情有独钟的话，那么这款笔记本是可以完成日常办公等使用场景的要求。

ThinkPad X1 Carbon 2017 最强生产力

在使用笔记本进行办公时，我们到底需要的是什么？是运行效率、安全和稳定性，还是键盘输入手感和机身轻薄？相信对于以上的特性，很多人会回答都需要。而专门为办公人士设计的商务本，也朝着这样的方向去努力。而在性能、体积和手感等方向做的最好的，无疑就是ThinkPad旗下如同传奇般存在的X1 Carbon系列。

选择ThinkPad X1 Carbon2017，肯定对其相当经典的家族式设计有着相当高的认同感。和市面上外观配色上相当“小鲜肉”的商务本相比，ThinkPadX1 Carbon棱角分明的设计语言，凸显其浓郁的专业气息。


配置上，ThinkPad X1 Carbon可选Intel Core i5-7200U和Intel Core i7-7500U处理器，内存可选8GB到16GB内存，储存空间最高可选择512GBSSD。单凭配置来说，ThinkPad X1 Carbon和其他热门商务本并没有太大的性能差距。不过在各项基础体验上，它能获得远超同类型笔记本的优势。


ThinkPad X1 Carbon的屏幕边框低至5mm，轻松塞下一块14英寸的1080P分辨率的显示屏。在维持机身重量轻至1.13kg的同时，还让厚度低至15.95mm。接口方面，提供了两个USB3.0接口、HDMI、两个雷电接口和一个3.5mm耳机孔。如果嫌弃这些接口还不够丰富，还可选配ThinkPad WiGig无线拓展坞，告别繁琐线材的办公方式和拓展更多外接设备数量。

ThinkPad X1 Carbon能够给你无限的可能，但售价上就不是很亲民。不过花上近W元的价格入手这个产品，你可以获得的是一台各方面表现都极为优秀的商务本。如此看来，ThinkPadX1 Carbon 2017的售价也并非不能接受。

总结：

上述的几款笔记本涵盖了不同价位区间，并拥有各自独特的亮点。对于有意购买商务本的消费者来说，事先确定好自己的预算区间和划分需求次序，是能不能挑选到心仪的商务本关键。不过要提醒一句的是，相比现在热门的手机而言，笔记本还是显得有点昂贵。所以在事先确定购买笔记本预算时，可别按照购买手机的思维去考量。
