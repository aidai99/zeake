耀晶记 - 索尼新品试听会武汉站 NW-A45 | NW-ZX300A | WF-1000X

71932 0
[attach]302062[/attach]

或许提起去年的 醇音典范 One Signature Sound 索尼高保真旗舰新品发布会, 许多人仍记忆犹新 - 覆盖 23 个城市, 共计 44 场试听会, 我曾以 "王者归来" 给予了 "金砖" NW-WM1Z 高度评价。

时隔一年, 9月9日, 武汉金索 再次举办新品试听会, 本次的主要器材是 NW-A45 播放器; NW-ZX300A 播放器 及 WF-1000X 耳塞 等几款新品。

[attach]302063[/attach]

活动地点定于 街道口 创意城 量子学园咖啡。

[attach]302075[/attach]

在完成简单的扫码与问卷调查后, 每人可领取一份免费饮品与甜点。 ( 试听则在旁边单独的会议室举办 )

[attach]302076[/attach]

[attach]302064[/attach]

SONY Walkman NW-ZX300A, 3.1 英寸 触摸屏 配合 侧面实体按键。

[attach]302065[/attach]

4.4mm 平衡口 及 3.5mm 接口, 关于输出功率 ( 推力 ) 想必不少烧友非常关心. ZX300A 4.4mm 接口为 200mW 16欧, 3.5mm 接口为 50mW 16欧 输出; 众所周知的 "金砖" 与 "黑砖" 4.4mm 接口输出为 250mW 16欧 及 3.5mm 接口 60mW 16欧。

从参数以及网传的定价 ( 3000RMB 上下 ) 来说, ZX300A 必将在市场定位上代替自家旧作 ZX100。

[attach]302066[/attach]

ZX300A 的系统网上并没有太多介绍, 从其逻辑及布局来看, 个人认为其仍然沿袭 "金砖" 与 "黑砖" 那套深度定制精简的 安卓系统。

光顾着拍照, 简单试听了下, 在 [ DSEE HX: 标准 / DC 相位线性器: A Low / 动态音量均衡器: 开启 设置下 ] , 4.4mm 接口下的 MDR-1A 素质不错, 经过这番设置后, 推力上 ZX300A 的表现确实值得肯定. 但与 4.4mm 换线的 N3AP 搭配, 我个人觉得稍有提升, 但幅度有限。

如果要说什么缺点, ZX300A 其 3.1 寸的触摸屏在这个掌设竞争异常激烈的年代, 配置稍显落伍, 机器下面大块的黑边并不美观。

[attach]302067[/attach]

SONY Walkman NW-A45 对比 ZX300A 大小。

[attach]302068[/attach]

NW-A15 / A25 / A35? 或许对于更多烧友而言, 大家还没理清楚这些旧型号的产品, 索尼又目不暇接的推出新作 A45。

没有详细检查 A45 的音效设置, 用随身带的 铁三角 ATH-CK100 简单试了一下这款新品, 总体感觉倒是 '音效' 不错, 播放器声音风格 "染色" 较多. ( 注此处与 "音质" 含义有别 ) 除去狂热的器材党, 我认为如果诸位的手机必须保持通信, 不能因听歌导致电量不足关机, 某种意义上来说 A45 算是这个掌设大行其道的年代夹缝生存的音频产品。

N45 样机固件版本为 1.00, 但系统流畅度上, A45 比 A35 有了明显提升, 同样基于 深度精简定制的 安卓系统。

[attach]302069[/attach]

SONY WF-1000X & NW-A45

[attach]302077[/attach]

产品还未正式发售, 与自家头戴 MDR-1000X 型号名相仿的 WF-1000X 就已吸引了无数目光。

遥想起数年前设计于主动降噪耳机线控上硕大的电池仓, 以及开启降噪功能后耳机阻抗大幅提升和轻微的压迫感, 那时的主动降噪耳机似乎一直是少数常年出差用户的选择. 而如今的技术却足以把 主动降噪, 环境音模式, 蓝牙 / NFC 这些功能做到小巧的耳塞上, WF-1000X 确实稍显 "耀眼"。

[attach]302070[/attach]

耳机盒为 USB 充电 ( 充电盒可为耳机充电两次 ), WF-1000X 有 金色 / 黑色两款

[attach]302071[/attach]

WF-1000X 支持通话 / 接听功能, 耳塞单元部分为 6mm 动圈。

由于自带手机并不支持 NFC / aptX 等功能, 对 WF-1000X 测试也稍显仓促: 以蓝牙模式连接, 新秀 WF-1000X 沿袭了自己比较熟悉的 MDR-1000X 主动降噪 及 "环境音" 功能, 抛开了那些 "控制变量法" ( aptX ) 声音水平上来说 WF-1000X 给我第一耳印象算是中规中矩。

[attach]302072[/attach]

( 量子学园咖啡 内景 )

[attach]302073[/attach]

[attach]302074[/attach]

旁边单独的会议室供烧友提供试听

总结

下午 2 点至 5 点, 三小时的试听活动稍显短暂...

从曾经的 Walkman NW-ZX2 到 "金砖" NW-WM1Z, 我看到了 SONY Walkman 品牌重回王位的野心, 但今天 NW-ZX300A 与 NW-A45, 抛开那些追求极致的发烧友, 索尼倒是开始重视起了中低端市场。

4.4mm 接口, 200mW 16欧 输出, 我认为 ZX300A 很快会接手自家 ZX100 的位置, 流畅度更佳的 A45 替代 A35 也将顺理成章, 而备受瞩目的 WF-1000X, 希望工业设计上乘且功能丰富的它能给市场带来更多冲击, 在无线主动降噪耳塞这类产品上, 也期盼其他厂商能推出更多有趣的新品。
