少女 EDC | off toco - Elecom 宜丽客 DGB-S038

30598 3


stuff / EDC 分享 及 出镜: @owo大野HANACO
摄影 & ACR 后期: @PairLens
拍摄地点: @莫奈花园法式甜点 ( 武汉天地店 )
摄影器材: 佳能 6D + 佳能 EF 35mm F/2 ( 产品由 佳能 EF 100mm F/2.8 拍摄 )
音频器材: 铁三角 ATH-ESW10JPN ( 越前漆 限量款 ) / SONY PCM-D50 / 乐彼 L3Pro

入秋后, 度过阴雨连绵的一周, 难得周日 ( 9月24号 ) 天气晴朗, 约了新麻豆在 黄浦路附近的 莫奈花园 ( 甜点 / 咖啡 ) 拍了套新片.

原计划在 Vessel Cafe 容器 在拍摄, 然而到店后, 除了满脸横肉让人作呕的店员, 稍显狭窄且采光并不理想的店铺外, 我实在是想不出这个曾在网上以拍照作为噱头, 实际又不让拍摄的小咖啡馆有什么优势可言......

从搭配, 风格, 线材到前端, 或许对于音频器材我有一套自己的观点, 但这次除了引言与产品介绍部分外, 我将退到幕后, 让麻豆完成这次有趣的 EDC ( Everyday Carry ) 分享.



莫奈花园 甜品





我个人感受是: 甜品稍腻, 咖啡略苦.



宜丽客 Elecom DGB-S038 摄影包

从官网摘抄关于 off toco 的简介:" 终于找到了休息日想带出门的相机包了, 对功能和设计都很讲究的我, 可以尽情融入大自然轻松携带, 想拍摄的时候可以立即取出相机拍摄, 平常散步或者短途小游也可以使用, 有 off toco 的陪伴, 轻松愉悦每一天.



上次接触 DGB-S033 单肩包早已为其内部 "迷宫式" 的设计所折服, 因其丰富的多级拉链分层, 初次使用的我甚至偶尔找不到存放在包里的产品.



拉开拉链, 顶部可以存放一个相机机身 与 一枚镜头. [ 佳能 100mm EF 100mm F/2.8 ]



[ 佳能 EF 35mm F/2 ] 另一侧倒亦能存放一台平板电脑.



存放钥匙, MP3 播放器或者一些硬币? 左边肩带上的小包都用拉链细分两隔断.



左右两侧的拉链开口应该是为各种瓶装饮料而备了.



宜丽客 DGB-S038 摄影包 有 黑色 与 灰色 两款
包内部尺寸为: ( 毫米 ) W270 x D85 x H390 / 重量: 1485 克

接下来由 狮子座 的麻豆 @owo大野HANACO 为大家带来 EDC 分享, 用她的私人物品来装满这个分层丰富的摄影包.

----------------------------------------------------------------------------------------------



爱丽小屋 心机唇釉 PK012 人鱼姬色
作为女生应该都无法抗拒亮闪闪的东西吧, 这款唇釉可以让少女心得到很大的满足, 也可以说是贵妇品牌的完美替代品, 涂在嘴上闪闪亮亮, 可爱非常.



资深堂 Integrate Gracy 完美意境保湿遮瑕提亮粉底霜 P010 粉调明亮色
这款粉底霜是我的底妆必备, 质感柔滑, 而且耐用, 它还能根据你的皮肤自行 "调色" 哦! 遮瑕力很棒, 持妆效果也是很不错的, 保湿度也非常厉害, 这个可以说是我的大宝贝儿了!



菲诗小铺 染眉膏3号灰棕色
老话说眉毛画得好, 福气少不了. 所以眉毛可以说是非常之重要的, 这一款染眉膏很是持久耐用3号色对于深系发色的人来说是非常的百搭了.



名创优品茉莉修护润手霜
天气渐渐转凉皮肤也会特别的干燥, 女生的手就是女生的第二张脸, 一定要好好保护, 这款润手霜气味清新, 质感清爽不油腻, 吸收快, 非常好用.



气味图书馆 阳光味吃酒淡香水
这款香水是之前买香水时送的一瓶小样, 这个味道很让我惊喜, 淡淡的还带着一丝墨香, 香味的持久度也是惊人, 一整天味道都会在, 不过是浓与淡的区别, 我通常喷在空中然后自己在香水里转个圈儿, 全身上下香喷喷, 心情也会很好.



名创优品 My Melody 合作款变色润唇膏
这个季节干燥的不止皮肤还有嘴唇, 这款润唇膏, 有着甜甜的味道, 涂在嘴上会慢慢的随着你的唇色变成漂亮的颜色的, 很滋润, 不同的唇色显示的颜色也会不一样哦, 是不是很神奇!



CANMAKE 井田棉花糖固妆控油蜜粉饼 ML色
这款粉饼物美价廉, 可以说是平价粉饼中的翘楚了, 粉质细腻, 控油效果极好, 可以打造出完美的哑光质感, 让人像一个瓷娃娃一样可爱, 个人比较喜欢让自己白一些所以选的也是最白的色号.

iPad Air 2 [ 16GB ]
没有追新跟进 IOS 11 系统, 仍在使用 IOS 10.33, 两年多的时光即使保护套磨损明显, Air2 仍是一如既往的流畅.



Dr.Althea 艾医生水光气垫 BB霜
气垫对我来说其实就是一个补妆小能手, 艾医生的这一款气垫 BB霜 可以说是很优秀了, 质感光滑水润, 让沙漠皮的我可以有走入绿洲疯狂喝水的快感, 这款气垫搭配棉花糖粉饼个人认为是绝配了.



博士伦 润明新概念 隐形眼镜润滑液
对于长时间佩戴隐形眼镜的小伙伴来说这个就是救命的东西, 干涩的眼睛又不能把隐形眼镜取下来用它来润润眼就非常便利了.



卡姿兰 零触感眼线笔
零触感无晕染, 一笔刷成, 笔毛柔软细腻; 睫毛膏也很好, 无苍蝇腿效果, 刷出来很密很长, 卡姿兰彩妆, 你值得拥有.



自然乐园 花漾超级睫毛膏 02号
这款睫毛膏可以让睫毛根根分明, 不是浮夸的大卷翘, 很日常, 上睫毛下睫毛都非常适用.



佳洁士 3D 炫白牙贴
因为长时间熬夜工作咖啡和茶可以说是离不开的东西了, 可是随之而来的就是日渐泛黄的牙齿了, 就连大笑都会不好意思, 这个牙贴却非常棒! 偶尔在工作或等车途中贴上半个小时后取下就会有暂白的效果, 用到一盒左右牙齿就会白好几度, 很是方便.



佳能 35mm F/2 IS USM 及 化妆品收纳包
摄影他比较偏好的一枚定焦镜头, 35mm 人文, 扫街, "可进可退" 大概是这个焦段的精髓, 3500 RMB 上下的价格, 硬要说啥短板, 就是光圈不够大.



佳能 100mm F/2.8 IS USM
这一期 EDC 分享文中的产品基本由这支定焦镜头拍摄, 100mm 微距虽然也能凑合拍人像, 但控制不好距离就会稍显脸大, 拍各类小玩意, 确实是名副其实的红圈镜头.



Oppo R7SM
这是一款不太成熟的 Oppo 手机, 为什么买它呢? 因为在它刚出来的时候性价比是真的高啊! 1500 RMB 出头的手机, 比 4000 RMB 多的 T8 好用到不知道哪里去了, 当然玫瑰金也是吸引我的最重要因素.



美图 T8
美图 T8 是网红产品, 它的人像自拍就跟宣传一样完全就是黑科技, 让爱自拍的女生成为 "网骗" 的利器, 玫瑰金的机身更是少女感爆棚!



铁三角 ATH-ESW10JPN ( 越前漆 限量款 )
2008 年底上市的 铁三角 ATH-ESW10JPN 便携头戴耳机如今保有量恐已屈指可数.



ESW10JPN 与 ESW9 等几款模具相似, 耳罩可相互通用.



如果我没记错的话, 除了 ESW10JPN 外, 用过越前漆的 铁三角 产品只剩 HA-5000 ANV 耳放 以及 ATH-W3000ANV 头戴耳机了.



尽管 ESW10JPN 动圈单元直径为 42mm, 但便携设计的它佩戴起来并不能完全包住耳朵.



45欧阻抗, 102dB 灵敏度, 最大输入功率 1000mW, 频响 5 ~ 35000Hz, 严格来说, ESW10JPN 并不算 "易推".



乐彼 L3Pro / 铁三角 ATH-ESW10JPN

----------------------------------------------------------------------------------------------































总结

让人目不暇接的分层, 携带舒适的 Elecom DGB-S038 不仅是一款适合短途旅行, 享受音乐, 携带各类数码产品的摄影包, 同样也是一款方便携带各类化妆品的 "大型收纳包".
